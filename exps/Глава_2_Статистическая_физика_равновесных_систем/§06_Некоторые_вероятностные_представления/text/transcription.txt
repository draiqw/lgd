§6. Некоторые вероятностные представления

══════════════════════════════════════════════════════════════════
КРАТКИЕ ТЕОРЕТИЧЕСКИЕ СВЕДЕНИЯ
══════════════════════════════════════════════════════════════════

В рамках данного пособия невозможно дать строгое определение математической вероятности и связанных с ней понятий. Необходимые сведения в полном объёме читатель может найти в соответствующих учебниках по математике. Ниже приведены лишь те понятия и формулы, которые будут активно использоваться в этом пособии.

Вероятностью события A называют отношение числа m, соответствующему этому событию реализаций, к полному числу n возможных попарно несовместных равновероятных реализаций: P(A) = m/n. Если в результате некоторого события величина x может принимать несколько различных значений, то она называется случайной. В дальнейшем будем рассматривать два типа таких величин.

Дискретная случайная величина может принимать конечное или бесконечное, но счётное число значений x₁, x₂, ..., xₙ, каждое из которых характеризуется своей вероятностью P(xᵢ). Зависимость P(xᵢ) в этом случае называется функцией распределения вероятностей дискретной случайной величины. При этом суммарная вероятность, как вероятность достоверного события, равна единице: Σᵢ P(xᵢ) = 1.

Непрерывная случайная величина x может принимать бесконечное и несчётное число значений из некоторого интервала (a, b). Для неё вводят плотность вероятности (или плотность распределения вероятностей) w⁽¹⁾(x), определяемую из условия, что w⁽¹⁾(x)dx есть вероятность того, что x попадает в интервал значений (x, x + dx).

В отличие от вероятности, являющейся безразмерной величиной, плотность распределения вероятностей имеет размерность, обратную x, а также удовлетворяет условию нормировки: ∫ₐᵇ w⁽¹⁾(x)dx = 1. Зависимость

    F₁(x̃) = ∫ₐˣ̃ w⁽¹⁾(x)dx                                             (6.1)

называют функцией распределения непрерывной случайной величины. При каждом x̃ она принимает значение, равное вероятности того, что x попадает в интервал (a, x̃).

Математическим ожиданием, или средним значением дискретной или непрерывной случайной величины x называют соответственно выражения:

    x̄ = Σᵢⁿ xᵢP(xᵢ),        x̄ = ∫ₐᵇ xw⁽¹⁾(x)dx.                       (6.2)

Среднее значение функции f(x) определяется с помощью соотношений:

    f̄(x) = Σᵢⁿ f(xᵢ)P(xᵢ),        f̄(x) = ∫ₐᵇ f(x)w⁽¹⁾(x)dx.           (6.3)

Дисперсия σ²(x) дискретной или непрерывной случайной величины x определяется формулами:

    σ²(x) = Σᵢⁿ (xᵢ − x̄)²P(xᵢ),        σ²(x) = ∫ₐᵇ (x − x̄)²w⁽¹⁾(x)dx.  (6.4)

Величину σ(x) называют средним квадратичным, или стандартным отклонением. Она показывает меру разброса x относительно x̄. Относительную флуктуацию случайной величины x определим как δ(x) = σ(x)/x̄.

Если задана функция y = y(x) и существует обратная функция x(y), а также известна плотность распределения вероятностей w⁽¹⁾(x) случайной величины x, то w⁽¹⁾(y) = w⁽¹⁾(x(y))|dx/dy|.

Совместная плотность распределения вероятностей (или совместная плотность вероятностей) w⁽ⁿ⁾(x₁, x₂, ..., xₙ) одновременно реализуемых случайных величин x₁, x₂, ..., xₙ находится из условия, что w⁽ⁿ⁾(x₁, x₂, ..., xₙ)dx₁dx₂...dxₙ — вероятность того, что x₁ попадает в интервал значений (x₁, x₁ + dx₁), x₂ — в (x₂, x₂ + dx₂) и т.д.

Среднее значение функции f(x₁, x₂, ..., xₙ) в этом случае определяется выражением

    f̄(x₁, x₂, ..., xₙ) = ∫...∫ w⁽ⁿ⁾(x₁, x₂, ..., xₙ)f(x₁, x₂, ..., xₙ)dx₁dx₂...dxₙ.   (6.5)

Здесь интегрирование проводится по всей области допустимых значений случайных величин x₁, x₂, ..., xₙ. Плотность распределения вероятностей одной из величин, например w⁽¹⁾(xₘ), легко найти, зная w⁽ⁿ⁾(x₁, x₂, ..., xₙ):

    w⁽¹⁾(xₘ) = ∫...∫ w⁽ⁿ⁾(x₁, x₂, ..., xₘ, ..., xₙ)dx₁dx₂...dxₘ₋₁dxₘ₊₁...dxₙ.   (6.6)

Зависимость

    Fₙ(x̃₁, x̃₂, ..., x̃ₙ) = ∫ₐ₁ˣ̃¹ ∫ₐ₂ˣ̃² ... ∫ₐₙˣ̃ⁿ w⁽ⁿ⁾(x₁, x₂, ..., xₙ)dx₁dx₂...dxₙ

называется совместной функцией распределения n непрерывных случайных величин. При любых x̃₁, x̃₂, ..., x̃ₙ она принимает значение, равное вероятности того, что одновременно x₁ попадает в интервал (a₁, x̃₁), x₂ в интервал (a₂, x̃₂) и т.д.

Пусть n случайных величин x₁, x₂, ..., xₙ однозначно связаны с n другими случайными величинами y₁, y₂, ..., yₙ формулами: yᵢ = yᵢ(x₁, x₂, ..., xₙ), где i = 1, 2, ..., n. Если известна совместная плотность распределения вероятностей w⁽ⁿ⁾(x₁, x₂, ..., xₙ), а также существуют обратные функции xᵢ = xᵢ(y₁, y₂, ..., yₙ), то

    w⁽ⁿ⁾(y₁, y₂, ..., yₙ) = w⁽ⁿ⁾(x₁(y₁, y₂, ..., yₙ), ..., xₙ(y₁, y₂, ..., yₙ)) × |∂(x₁, x₂, ..., xₙ)/∂(y₁, y₂, ..., yₙ)|.   (6.7)

Последний множитель в (6.7) — якобиан преобразования от переменных x₁, x₂, ..., xₙ к переменным y₁, y₂, ..., yₙ.

Пусть событие A может наступить при условии появления одного из n попарно несовместных событий B₁, B₂, ..., Bₙ, образующих полную группу возможных событий. Тогда вероятность появления события A задаётся формулой полной вероятности

    P(A) = P(B₁)P(A|B₁) + P(B₂)P(A|B₂) + ... + P(Bₙ)P(A|Bₙ),          (6.8)

где P(A|Bᵢ) — вероятность наступления события A при реализации события Bᵢ.

Условной плотностью распределения вероятностей w(y|x) случайной величины y при фиксированном значении x называют функцию, удовлетворяющую равенству w⁽¹⁾(x)w(y|x) = w⁽²⁾(x, y). Если w(y|x) не зависит от x, то x и y статистически независимы. В общем случае случайные величины x₁, x₂, ..., xₙ называются независимыми, если совместная плотность распределения вероятностей равна произведению плотности вероятности каждой из них

    w⁽ⁿ⁾(x₁, x₂, ..., xₙ) = w⁽¹⁾(x₁)w⁽¹⁾(x₂)...w⁽¹⁾(xₙ).               (6.9)

Из (6.9) легко получить, что для двух независимых случайных величин x₁ и x₂ справедливо равенство x̄₁x₂ = x̄₁ · x̄₂.

Пример 6.1. Плотность вероятности попадания частицы в любую точку плоскости, находящуюся внутри некоторой области, имеющей площадь S, является постоянной величиной. Вычислить условную вероятность того, что частица оказется внутри прямоугольного треугольника с углом α при вершине, если известно, что она находится внутри описанной вокруг него окружности, полностью лежащей внутри вышеупомянутой области.
───────────────────────────────────────────────────────────────────

Решение. Назовём событием A попадание частицы в круг, радиус которого R, а событием B — в область, ограниченную вписанным в него треугольником. Их вероятности P(A) = πR²/S и P(B) = 2R² cos α sin α/S. В соответствии с формулой (6.8) P(B) = P(A)P(B|A), откуда P(B|A) = P(B)/P(A) = sin 2α/π.

Пример 6.2. В начальный момент t = 0 в области пространства, имеющей форму параллелепипеда, одно из рёбер которого равно a, находится N₀ одинаковых частиц, летящих с постоянной скоростью v⃗, направленной параллельно упомянутому ребру (см. рис. 6.1). При t = 0 плотность распределения вероятностей нахождения частиц имеет вид: w⁽³⁾(x, y, z) = w⁽¹⁾(x)w⁽²⁾(y, z). Здесь

    w⁽²⁾(y, z) = { 1/S,  y, z ∈ S
                { 0,    y, z ∉ S

    w⁽¹⁾(x) = { 0,           x < b
             { 2(x − b)/a²,  b + a ≥ x ≥ b                            (6.10)

S — площадь боковой грани параллелепипеда, перпендикулярной вектору v⃗. Ось 0x декартовой системы координат направлена вдоль ребра, имеющего длину a. График зависимости w⁽¹⁾(x) изображён на рис. 6.1. В точке x = 0 расположена перпендикулярная оси 0x стенка, расстояние от которой до ближайшей грани параллелепипеда в начальный момент времени равно b. Найти зависимости от времени числа частиц N(t), ударившихся в стенку, а также давления p(t), которое они оказывают. Соударения между частицами происходят крайне редко. Масса одной частицы m₀. Удар считать абсолютно упругим.
───────────────────────────────────────────────────────────────────

Решение. Удовлетворяющая условию нормировки плотность вероятности того, что в момент времени τ частица облака долетела до стенки, легко находится из (6.10)

    w̃⁽¹⁾(τ) = { 0,                  τ < b/v
             { 2v(vτ − b)/a²,       (b + a)/v ≥ τ ≥ b/v.

За время dt о стенку ударяется N₀w̃⁽¹⁾(τ)dt частиц. Полное число частиц, ударившихся о стенку за время t,

    N(t) = N₀ ∫₀ᵗ w̃⁽¹⁾(τ)dτ = { 0,                          t < b/v
                              { N₀(vt − b)²/a²,  (b + a)/v ≥ t ≥ b/v.

График зависимости N(t) приведён на рис. 6.2.

По условию задачи удар о стенку абсолютно упругий. Передаваемый частицами за время dt импульс 2m₀vN₀w̃⁽¹⁾(τ)dt равен импульсу силы p(τ)Sdt, откуда p(τ) = 2m₀vN₀w̃⁽¹⁾(τ)/S.

Пример 6.3. Нормальный закон распределения (закон Гаусса) случайной величины x задаётся плотностью вероятности

    w⁽¹⁾(x) = (1/√(2πσ)) exp(−(x − x₀)²/2σ²).                         (6.11)

Определить наиболее вероятное значение x*, среднее значение x̄ и относительную флуктуацию δ(x) случайной величины x.
───────────────────────────────────────────────────────────────────

Решение. Наиболее вероятное значение случайной величины находим из условия (∂w⁽¹⁾/∂x)|_{x*} = 0, откуда x* = x₀. В соответствии с (6.2), (6.4) получим:

    x̄ = (1/√(2πσ)) ∫_{-∞}^{∞} x exp(−(x − x₀)²/2σ²)dx = x₀,         (6.12)

    σ²(x) = (1/√(2πσ)) ∫_{-∞}^{∞} (x − x₀)² exp(−(x − x₀)²/2σ²)dx = σ².   (6.13)

Относительная флуктуация δ(x) = σ(x)/x̄ = σ/x₀.

На рис. 6.3.а изображены графики зависимостей w⁽¹⁾(x) при фиксированном значении σ. Кривые 1, 2 соответствуют x₀ = 0 и x₀ > 0. На рис. 6.3.б приведены графики зависимостей w⁽¹⁾(x) при x₀ = 0. Дисперсия распределения, показанного кривой 2, в два раза больше, чем дисперсия распределения, которому соответствует кривая 1. Видно, что увеличение дисперсии приводит к уширению графика w⁽¹⁾(x) и уменьшению максимального значения функции. При этом площадь под кривой w⁽¹⁾(x) остаётся равной единице в соответствии с условием нормировки.

Пример 6.4. N молекул газа находятся в сосуде объёмом V. Найти функцию распределения вероятностей P_N(n) случайного события, когда n частиц окажутся в малой части сосуда объёмом v << V. Вычислить среднее значение, дисперсию и относительную флуктуацию n.
───────────────────────────────────────────────────────────────────

Решение. Вероятность молекуле попасть в объём v равна v/V. Вероятность того, что в этот объём попадёт n молекул, равна (v/V)ⁿ. Вероятность всем остальным молекулам не оказаться в объёме v равна (1 − v/V)^(N−n). Учитывая возможность перестановок между одинаковыми частицами, окончательно получим

    P_N(n) = C_N^n (v/V)ⁿ(1 − v/V)^(N−n) = N!(v/V)ⁿ(1 − v/V)^(N−n)/n!(N − n)!,   (6.14)

где n = 0, 1, 2, ..., N. Пользуясь формулой (6.3) получим среднее значение n

    n̄ = Σₙ₌₀^N (nN!/(N − n)!n!)(v/V)ⁿ(1 − v/V)^(N−n) = (Nv/V) Σₙ₌₁^N ((N−1)!/(n−1)!(N−n)!)(v/V)^(n−1)(1 − v/V)^(N−n) = (Nv/V) Σₖ₌₀^(N−1) C_{N−1}^k (v/V)^k(1 − v/V)^(N−1−k) = Nv/V.

При записи последнего равенства мы учли, что распределение P_N(n) удовлетворяет условию нормировки Σₙ₌₀^N P_N(n) = Σₖ₌₀^N C_N^k (v/V)^k(1 − v/V)^(N−k) = 1.

Таким образом, молекулы идеального газа равномерно распределены по всему объёму. Пользуясь (6.4), найдём дисперсию случайной величины n

    σ²(n) = Σₙ₌₀^N (n − Nv/V)²P(n) = Σₙ₌₀^N n(n−1)P(n) + (1 − 2Nv/V) Σₙ₌₀^N nP(n) + (Nv/V)² Σₙ₌₀^N Pₙ.

При записи последней формулы n² было представлено в виде n(n − 1) + n. Заменив, как и ранее, индекс суммирования, окончательно получим

    σ²(n) = (v/V)²N(N − 1) + (N − 2N²v/V)v/V + N²(v/V)² = (1 − v/V)Nv/V.   (6.15)

Наконец, относительная флуктуация числа частиц в объёме v будет иметь вид: δ = √(V/v − 1)/√N. Таким образом, с ростом N среднее число частиц в объёме v увеличивается пропорционально N, дисперсия возрастает как N, а δ уменьшается пропорционально 1/√N. График функции распределения вероятностей P_N(n) изображён на рис. 6.4 (v/V = 0,5) и рис. 6.5 (v/V = 0,1) при N = 8 (а) и N = 100 (б). Дисперсия относительного числа частиц уменьшается с ростом v/V. Это иллюстрирует рис. 6.6, где приведена зависимость P₁₀₀ от относительного числа частиц n/n̄ при v/V = 0,5 и v/V = 0,1.

Если N >> 1, а v << V, то P_N(n) стремится к распределению Пуассона: P_N(n) ≈ ((Nv/V)ⁿ/n!)·exp(−Nv/V). В окрестности максимума последнюю формулу можно с высокой точностью аппроксимировать распределением Гаусса

    P_N(n) = (1/√(2π)) · (1/√((1 − v/V)Nv/V)) · exp[−(n − Nv/V)²/(2(1 − v/V)Nv/V)].   (6.16)

Это хорошо видно на рис. 6.7, где показаны зависимости P_N(n) при N = 100 и v/V = 0,045 (а) и v/V = 0,25 (б).
