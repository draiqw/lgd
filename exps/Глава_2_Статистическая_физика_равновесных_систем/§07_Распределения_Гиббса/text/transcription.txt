§7. Распределения Гиббса

══════════════════════════════════════════════════════════════════
КРАТКИЕ ТЕОРЕТИЧЕСКИЕ СВЕДЕНИЯ

В момент времени t состояние термодинамической системы, состоящей из N одинаковых бесструктурных частиц, подчиняющихся законам классической механики, определяется заданием значений радиус-векторов r⃗₁(t), r⃗₂(t), ..., r⃗ₙ(t) и импульсов p⃗₁(t), p⃗₂(t), ..., p⃗ₙ(t) всех частиц системы. Для краткости будем использовать обозначение xᵢ = (r⃗ᵢ, p⃗ᵢ), где i = 1, 2, ..., N, для совокупности значений координат и компонент импульса отдельной частицы. Совокупность значений координат и компонент импульсов всех частиц, определяющих микросостояние системы, обозначим X, т.е. X = (x₁, x₂, ..., xₙ) = (r⃗₁, r⃗₂, ..., r⃗ₙ, p⃗₁, p⃗₂, ..., p⃗ₙ).

Зависимость X = X(X₀, t, t₀), где X₀ — состояние системы в начальный момент времени t₀, описывает временную эволюцию системы. Для нахождения X = X(X₀, t, t₀) используют уравнения движения. Запишем их в форме уравнений Гамильтона:

    dr⃗ᵢ/dt = ∂H(X)/∂p⃗ᵢ,     dp⃗ᵢ/dt = −∂H(X)/∂r⃗ᵢ,                    (7.1)

где функция Гамильтона имеет вид:

    H(X) = Σᵢ₌₁ᴺ p⃗ᵢ²/2m + U(r⃗₁, ..., r⃗ₙ) = Σᵢ₌₁ᴺ (p⃗ᵢ²/2m + U₀(r⃗ᵢ)) + Σᵢ<ⱼ Φ(|r⃗ᵢ − r⃗ⱼ|).   (7.2)

В (7.2) m — масса частицы, U₀ — потенциальная энергия частицы во внешнем поле, Φ(|r⃗ᵢ − r⃗ⱼ|) — потенциальная энергия взаимодействия частиц с номерами i и j.

При исследовании движения частиц в макроскопических системах, задаваемых совокупностью внешних параметров a = (a₁, a₂, ...aₙ), нет возможности задать начальные условия для всех частиц, а, следовательно, и решить систему (7.1). Однако можно считать, что в каждый момент времени x₁, x₂, ..., xₙ являются случайными величинами, и интересоваться функцией распределения возможных значений X. Будем рассматривать только такие случаи задания термодинамической системы, когда плотность распределения вероятностей микросостояний wₙ(x₁, x₂, ..., xₙ, t, a) ≡ wₙ(X(t), t, a) в любой момент времени t удовлетворяет уравнению Лиувилля

    ∂wₙ/∂t + Σᵢ₌₁ᴺ (pᵢ/m · ∂wₙ/∂r⃗ᵢ − ∂U₀/∂r⃗ᵢ · ∂wₙ/∂p⃗ᵢ − ∂/∂r⃗ᵢ Σⱼ₌₁ᴺ Φ(|r⃗ᵢ − r⃗ⱼ|) · ∂wₙ/∂p⃗ᵢ) = 0.   (7.3)

Для изолированной (замкнутой) системы или системы в постоянном внешнем поле состояние называется равновесным, если wₙ не зависит явно от времени и начальных условий. Энергетически изолированная от окружающих тел термодинамическая система с неизменным числом частиц имеет постоянную энергию E. В этом случае решение (7.3) при ∂wₙ/∂t = 0 имеет вид

    wₙ(X(t), a, E) = δ(H(X, a) − E)/Ω̃(a, E).                           (7.4)

Формула (7.4) называется микроканоническим распределением Гиббса. Оно удовлетворяет условию нормировки ∫wₙ(X(t), a, E)dX = 1. Благодаря наличию в ней δ-функции из всех возможных значений X выделяются лишь те, которые совместимы с условием H(X, a) = E. Как видно из (7.4), все эти значения равновероятны. Константу Ω̃(a, E) = ∫δ(H(X, a) − E)dX называют статистическим весом.

Для термодинамической системы, обменивающейся энергией с окружающими телами, число частиц в которых N₁ >> N (система в термостате), плотность распределения вероятностей микросостояний задаётся каноническим распределением Гиббса

    wₙ(X(t), a, T) = exp(−H(X, a)/kT)/Z(a, T).                          (7.5)

Здесь kT = (2/3) lim_{N₁→∞} N₁⁻¹ Σᵢ₌₁^{N₁} p⃗ᵢ²/2m₁ — предел, к которому стремится средняя кинетическая энергия поступательного движения частиц, окружающих рассматриваемую термодинамическую систему, а Z(a, T) = ∫exp(−H(X, a)/kT)dX — классический статистический интеграл.

Каноническое распределение Гиббса также удовлетворяет условию нормировки. Если константу k в последней формуле выбрать равной постоянной Больцмана, то T совпадает с абсолютной термодинамической температурой.

Если термодинамическая система может быть разделена на m макроскопических частей, взаимодействие между которыми пренебрежимо мало, то гамильтониан приближённо можно представить в виде H(X) = Σ_{α=1}^m Hα(Xα), где Hα(Xα) — гамильтониан подсистемы с номером α. При этом (7.5) может быть представлена в виде произведения канонических распределений Гиббса для отдельных частей

    wₙ(X) = ∏_{α=1}^m exp(−Hα(Xα)/kT)/Zα(a, T, Nα),                    (7.6)

где Zα(a, T) = ∫exp(−Hα(Xα)/kT)dXα. Микроканоническое распределение Гиббса таким свойством не обладает.

В квантовой теории равновесного состояния также существуют два основных распределения: микроканоническое и каноническое распределение Гиббса, которые запишем ниже по аналогии с (7.4) и (7.5). Пусть теперь n = (n₁, n₂, ..., nₙ) — полный набор квантовых чисел, характеризующих одно из микроскопических состояний макроскопической системы из N частиц, а Eₙ — соответствующее этому набору значение её энергии. Последнее для равновесной системы может быть найдено в результате решения стационарного уравнения Шредингера.

Будем считать Eₙ известным для каждого отдельного набора квантовых чисел, если соответствующее ему состояние системы может реализоваться. Задаваемая совокупностью внешних параметров a термодинамическая система случайным образом может находиться в одном из квантовых состояний, описываемых одним из возможных наборов квантовых чисел n.

Функция распределения микросостояний Pₙ(N, a) системы с постоянным числом частиц N определяет вероятность того, что микроскопическое состояние характеризуется набором квантовых чисел n. Условие нормировки для функции Pₙ имеет вид Σₙ Pₙ(N, a) = 1. Заметим, что суммирование в последней формуле проводится по всем возможным наборам квантовых чисел.

Квантовое микроканоническое распределение Гиббса, справедливое для энергетически изолированной системы, имеет вид:

    Pₙ(N, a, E) = δ_{E,Eₙ}/Ω₁(a, N, E),                                  (7.7)

где δ_{E,Eₙ} — символ Кронекера, а Ω₁(a, N, E) = Σₙ δ_{E,Eₙ} — статистический вес, равный числу микросостояний с заданным значением энергии E. Из формулы (7.7) видно, что все микросостояния, удовлетворяющие условию Eₙ(a, N) = E, имеют одинаковую вероятность 1/Ω₁.

Функция распределения Pₙ(N, a, T) для квантовой системы в термостате имеет вид

    Pₙ(N, a, T) = exp(−Eₙ/kT)/Z₁(a, N, E) = exp((F − Eₙ)/kT),          (7.8)

где статистическая сумма

    Z₁ = Σₙ exp(−Eₙ/kT)                                                 (7.9)

связана со свободной энергией соотношением

    F = −kT ln Z₁(a, N, E).                                             (7.10)

При предельном переходе к системе, состоящей из N бесструктурных частиц, подчиняющихся законам классической механики, в формулах (7.7)−(7.9) Eₙ(a, N) надо заменить на H(X, a), а переход от статистической суммы Z₁ к статистическому интегралу осуществить следующим образом:

    Z₁ = Σₙ exp(−Eₙ/kT) → ϑ(N)Z(a, T),                                  (7.11)

где ϑ(N) = [(2πℏ)³ᴺ N!]⁻¹, ℏ = 1,0546·10⁻³⁴ Дж·с — постоянная Планка.

Если для находящейся в термостате системы заданным является не число частиц N, а химический потенциал μ, то плотность распределения микросостояний задаётся большим каноническим распределением Гиббса:

    P_{n,N}(a, T, μ) = exp[(μN − E_{n,N})/kT]/Z₂(a, T, μ).              (7.12)

В формуле (7.12) случайными величинами являются число частиц N и набор квантовых чисел n, однозначно задающих энергию термодинамической системы E_{n,N}. Функция P_{n,N}(a, T, μ) даёт вероятность того, что реализуется микроскопическое состояние, характеризующееся набором квантовых чисел n в тот момент, когда система состоит из N частиц. При этом статистическая сумма

    Z₂ = Σ_{N,n} exp[(μN − E_{n,N})/kT]                                 (7.13)

связана с большим термодинамическим потенциалом Ω(a, T, μ) = F − μN соотношением

    Ω = −kT ln Z₂(a, N, E) = −kT ln(Σ_{N,n} exp[(μN − E_{n,N})/kT]).    (7.14)

В (7.13), (7.14) суммирование производится по всем возможным наборам квантовых чисел при каждом значении числа частиц. Распределение (7.12) удовлетворяет условию нормировки Σ P_{n,N} = 1 и, с учётом (7.14), может быть записано в виде

    P_{n,N}(a, T, μ) = exp((Ω + μN − E_{n,N})/kT).                       (7.15)

Среднее число частиц N(a, T, μ) в термодинамической системе при заданных значениях a, T и μ можно выразить с помощью Ω(V, T, μ):

    N(a, T, μ) = −(∂Ω/∂μ)_{a,T}.                                        (7.16)

При предельном переходе к системе, состоящей из N бесструктурных частиц, подчиняющихся законам классической физики, распределение (7.12) переходит в выражение

    wₙ(X(t), N, a, T, μ) = exp[(μN − H(X, N, a))/kT]/Z̃₂(a, T, μ),      (7.17)

где случайными величинами являются X и N, значения которых определяются параметрами a, T и μ, а

    Z̃₂ = Σₙ ϑ(N) ∫exp[(μN − H(X, N, a))/kT]dX.                         (7.18)

Для построения термодинамики многочастичной системы достаточно определить один из её термодинамических потенциалов. Если её состояние определяется независимыми макроскопическими параметрами E, V и N, то термодинамическим потенциалом является энтропия, статистическое определение которой в литературе не является единственным. Ниже будем считать, что она связана с функциями распределения микросостояний (7.12) или (7.17) соотношениями:

    S(a, T, μ) = −k Σ_{n,N} P_{n,N} ln P_{n,N},                          (7.19)

    S(a, T, μ) = −k Σₙ ϑ(N) ∫wₙ ln(wₙ)dX = −k ln wₙ.                   (7.20)

Определённая формулами (7.19), (7.20) энтропия полностью удовлетворяет обсуждавшимся ранее термодинамическим закономерностям.

Выражения (7.19) и (7.20) обладают рядом свойств, определяющих их выбор в качестве меры неопределённости при статистическом описании рассматриваемых систем. Например, при полной определённости в задании микросостояний системы энтропия обращается в ноль, а при максимальной неопределённости в задании системы (равной вероятности всех микросостояний) достигает максимального значения. Ограничения на возможные значения случайных величин могут уменьшить (либо оставить неизменной) статистическую неопределённость. Эти и другие свойства делают S(a, T, μ) аналогичной энтропии, используемой в теории информации и в качестве меры неопределённости. Тем самым возможна статистическая трактовка второго начала термодинамики. В состоянии равновесия энтропия достигает своего максимального значения, поскольку равновесие является наиболее вероятным состоянием, определяемым фиксированными внешними условиями. Изолированная система, предоставленная самой себе, очевидно, движется в направлении равновесного состояния. При этом её энтропия должна возрастать. В неизолированных системах энтропия может (в зависимости от характера происходящего процесса) и возрастать, и убывать, и оставаться неизменной.

Пользуясь каноническим распределением Гиббса, можно доказать теорему о равномерном распределении кинетической энергии по степеням свободы. Она утверждает, что для классической системы, находящейся в состоянии термодинамического равновесия, среднее значение кинетической энергии, приходящейся на какую-либо степень свободы, равно kT/2. Если потенциальная энергия удовлетворяет равенству U(λr⃗₁, λr⃗₂, ..., λr⃗ₙ) = λⁿU(r⃗₁, r⃗₂, ..., r⃗ₙ), то среднее значение гамильтониана (7.2) определяется формулой

    H̄ = (s₁/2 + s₂/η)kT,                                                (7.21)

где s₁ — полное число степеней свободы, а s₂ — число колебательных степеней свободы.

Пример 7.1. Для идеального одноатомного газа записать плотность распределения вероятностей w₁(r⃗, p⃗) для координат и импульсов одной молекулы. Газ состоит из N молекул, находится в сосуде объёмом V и имеет температуру T. Масса молекулы m.
───────────────────────────────────────────────────────────────────

Решение. Для идеального одноатомного газа, находящегося в сосуде объёмом V, функция Гамильтона (7.2) принимает вид: H(X) = Σᵢ₌₁ᴺ (pᵢ²/2m + U₀(r⃗ᵢ)), где U₀(r⃗ᵢ) = 0, если r⃗ᵢ ∈ V, и U₀(r⃗ᵢ) = ∞, если r⃗ᵢ ∉ V. Используя формулу

    ∫_{-∞}^{∞} u²ⁿ exp(−αu²)du = (2n−1)!!/(2α)ⁿ √(π/α),                (7.22)

где α > 0, а (2n−1)!! равно 1·3·5·...·(2n−1) при n ≥ 1 и единице при n = 0, вычислим классический статистический интеграл

    Z(a, T) = ∫exp(−H(X, a)/kT)dX = ∫exp(−Σᵢ₌₁ᴺ pᵢ²/2mkT) dp⃗₁dp⃗₂...dp⃗ₙ ∫dr⃗₁dr⃗₂...dr⃗ₙ =
           = Vᴺ [∫∫∫exp(−(p²ᵢₓ + p²ᵢᵧ + p²ᵢᵤ)/2mkT) dpᵢₓdpᵢᵧdpᵢᵤ]ᴺ = Vᴺ(2πmkT)^{3N/2}.

Искомая функция имеет вид:

    w₁(xᵢ) = Z⁻¹(a, T) ∫exp(−H(X, a)/kT)dx₁dx₂...dxᵢ₋₁dxᵢ₊₁...dxₙ =     (7.23)
           = V^{N−1}(2πmkT)^{3(N−1)/2}/[Vᴺ(2πmkT)^{3N/2}] · exp(−p⃗ᵢ²/2mkT) = exp(−p⃗ᵢ²/2mkT)/[V(2πmkT)^{3/2}].

Заметим, что распределение частиц в пространстве ∫w₁(r⃗, p⃗)dp⃗ = 1/V равномерно внутри объёма V, а распределение частиц по импульсам

    w₁(p⃗) = w⁽³⁾(pₓ, pᵧ, pᵤ) = ∫w₁(r⃗, p⃗)dr⃗ = 1/(2πmkT)^{3/2} exp(−(p²ₓ + p²ᵧ + p²ᵤ)/2mkT)

является трёхмерным распределением Гаусса.

───────────────────────────────────────────────────────────────────
Пример 7.2. Пользуясь каноническим распределением Гиббса доказать формулу (7.21) при s₁ = 3N и η = 2, т.е. H̄ = 3NkT.
───────────────────────────────────────────────────────────────────

Решение. Пусть гамильтониан термодинамической системы из N частиц H = Σᵢ₌₁ᴺ Σ_{α=1}³ (p²ᵢα/2m + Aᵢαq²ᵢα/2), где i принимает значения 1, 2, ..., N, а α — 1, 2, 3, или x, y, z, а Aᵢα — положительные константы (по дважды встречающимся индексам суммирование не производится), тогда p̄ᵢα ∂H/∂pᵢα = q̄ᵢα ∂H/∂qᵢα = kT.

По определению среднего значения

    p̄ᵢα ∂H/∂pᵢα = ϑ(N)/Z₁ ∫pᵢα ∂H/∂pᵢα exp(−H(X)/kT) ∏ⱼ₌₁ᴺ dqⱼₓdqⱼᵧdqⱼᵤdpⱼₓdpⱼᵧdpⱼᵤ,   (7.24)

где ϑ(N) = [(2πℏ)³ᴺ N!]⁻¹, а

    Z₁ = ϑ(N) ∫exp(−Σⱼ₌₁ᴺ Σ_{β=1}³ (p²ⱼβ/2mkT + Aⱼβq²ⱼβ/2kT)) ∏ⱼ₌₁ᴺ dpⱼₓdpⱼᵧdpⱼᵤdqⱼₓdqⱼᵧdqⱼᵤ =   (7.25)
      = ϑ(N)(2πmkT)^{3N/2} ∏ⱼ₌₁ᴺ ((2πkT)³/(AⱼₓAⱼᵧAⱼᵤ))^{1/2}.

При вычислении интеграла в (7.25) мы воспользовались формулой (7.22), в которой положили n = 0.

Подставляя (7.25) в (7.24) и интегрируя по всем переменным кроме pᵢα, получаем:

    p̄ᵢα ∂H/∂pᵢα = (2πmkT)⁻¹/² ∫_{-∞}^{∞} (p²ᵢα/m) exp(−p²ᵢα/2mkT) dpᵢα.   (7.26)

Последний интеграл также легко вычислить с помощью (7.22) (n = 1):

    ∫_{-∞}^{∞} (p²ᵢα/mkT) exp(−p²ᵢα/2mkT) dpᵢα = (2πmkT)^{1/2}.          (7.27)

Подставляя (7.27) в (7.26), получаем искомое равенство p̄ᵢα ∂H/∂pᵢα = kT. Формула q̄ᵢα ∂H/∂qᵢα = kT доказывается аналогично. Среднее значение гамильтониана рассматриваемой термодинамической системы

    H̄ = Σᵢ₌₁ᴺ Σ_{α=1}³ (p̄²ᵢα/2m + Aᵢαq̄²ᵢα/2) = (1/2) Σᵢ₌₁ᴺ Σ_{α=1}³ [p̄ᵢα ∂H/∂pᵢα + q̄ᵢα ∂H/∂qᵢα] = 3NkT,

что и требовалось доказать.

───────────────────────────────────────────────────────────────────
Пример 7.3. Найти внутреннюю энергию одного моля идеального двухатомного газа, модель которого представляет молекулы, насаженные на жёсткую спицу и связанные пружиной с жёсткостью κ с двумя материальными точками, массы которых m₁ и m₂ (см. рис. 7.1). Молекулы могут свободно перемещаться в трёхмерном пространстве. Деформацию пружины считать малой.
───────────────────────────────────────────────────────────────────

Решение. Положение i-ой молекулы в пространстве задаётся тремя декартовыми координатами x_{i1}, y_{i1}, z_{i1} материальной точки массой m₁ и тремя декартовыми координатами x_{i2}, y_{i2}, z_{i2} материальной точки массой m₂, образующей молекулу с номером i. Если их импульсы p⃗_{i1} и p⃗_{i2}, то полная механическая энергия этой молекулы

    Eᵢ = (p²_{i1x} + p²_{i1y} + p²_{i1z})/2m₁ + (p²_{i2x} + p²_{i2y} + p²_{i2z})/2m₂ + Ũ,   (7.28)

где Ũ(x_{i1}, y_{i1}, z_{i1}, x_{i2}, y_{i2}, z_{i2}) — потенциальная энергия сжатой пружины. Выразим x_{i1,2}, y_{i1,2}, z_{i1,2} через координаты центра масс молекулы xᵢ, yᵢ, zᵢ, два угла θᵢ и φᵢ, характеризующих ориентацию спицы в пространстве (см. рис. 7.2), и деформацию пружины ξᵢ = lᵢ − l_{i0}. Здесь lᵢ — длина пружины, соответствующая координатам x_{i1,2}, y_{i1,2}, z_{i1,2}, l_{i0} — длина недеформированной пружины.

Декартовы координаты материальных точек m₁ и m₂ связаны с xᵢ, yᵢ, zᵢ, θᵢ и φᵢ и ξᵢ следующими соотношениями:

    x_{i1,2} = xᵢ ∓ (l₀ + ξᵢ)m_{2,1}(m₁ + m₂)⁻¹ sin θᵢ cos φᵢ,
    y_{i1,2} = yᵢ ∓ (l₀ + ξᵢ)m_{2,1}(m₁ + m₂)⁻¹ sin θᵢ sin φᵢ,           (7.29)
    z_{i1,2} = zᵢ ∓ (l₀ + ξᵢ)m_{2,1}(m₁ + m₂)⁻¹ cos θᵢ.

Подставив (7.29) в (7.28), получим

    Eᵢ = (m₁ + m₂)(ẋᵢ² + ẏᵢ² + żᵢ²)/2 + κξᵢ²/2 + m₁m₂[ξ̇ᵢ² + (l₀ + ξᵢ)²(sin²θᵢ φ̇ᵢ² + θ̇ᵢ²)]/(2(m₁ + m₂)).

Правая часть последнего выражения совпадает с функцией Гамильтона Hᵢ молекулы с номером i, если входящие в неё производные ẋᵢ, ẏᵢ, żᵢ, θ̇ᵢ, φ̇ᵢ и ξ̇ᵢ выразить через обобщённые импульсы:

    p_{iξ} = m₁m₂ξ̇ᵢ/(m₁ + m₂),    p_{iθ} = (l₀ + ξᵢ)²m₁m₂θ̇ᵢ/(m₁ + m₂),    (7.30)
    p_{iφ} = (l₀ + ξᵢ)²m₁m₂θᵢ/(m₁ + m₂).

С учётом (7.30) функция Гамильтона имеет вид

    Hᵢ = (p²ᵢₓ + p²ᵢᵧ + p²ᵢᵤ)/(2(m₁ + m₂)) + κξᵢ²/2 + (m₁ + m₂)/(2m₁m₂) [p²_{iξ} + (p²_{iθ} + p²_{iφ}/sin²θᵢ)/l₀²].   (7.31)

При записи (7.31) мы учли, что ξᵢ << l₀.

Воспользуемся теперь формулой (7.21). Каждая молекула имеет шесть степеней свободы, а всего их N_A, где N_A — число Авогадро. Поэтому s₁ = 6N_A. Из шести степеней свободы одна — колебательная и, следовательно, s₂ = N_A. Потенциальная энергия κ(λξᵢ)²/2 = λ²κξᵢ²/2 удовлетворяет равенству, поэтому η = 2. В итоге получим:

    H = Σᵢ₌₁^{N_A} H̄ᵢ = N_A(s₁/2 + s₂/η)kT = 7N_AkT/2 = 7RT/2.

───────────────────────────────────────────────────────────────────
Пример 7.4. Найти уравнение состояния, внутреннюю энергию, энтропию и химический потенциал ν молей одноатомного идеального газа. Число молекул N, масса одной молекулы равна m.
───────────────────────────────────────────────────────────────────

Решение. Для идеального одноатомного газа, находящегося в сосуде объёмом V, классический статистический интеграл

    Z(a, T) = ∫exp(−H(X, a)/kT)dx₁dx₂...dxₙ = Vᴺ(2πmkT)^{3N/2}

(см. пример 7.1). В соответствии с (7.10), (7.11) F = −kT ln Z₁, где Z₁ = [(2πℏ)³ᴺ N!]⁻¹Z и, следовательно,

    F = −kTN{1 + ln[V(2πmkT)^{3/2}/N(2πℏ)³]}.                           (7.32)

При записи последнего выражения была использована формула Стирлинга N! ∼ Nᴺ exp(−N).

Внутренняя энергия связана со свободной энергией равенством (3.10): U = F − T(∂F/∂T)_V. Подставляя F в последнюю формулу, получим: U = 3kTN/2.

Пользуясь формулой p = −(∂F/∂V)_T, найдём давление p = −(∂F/∂V)_T = kTN/V, откуда pV = νRT. С помощью формулы (3.6) найдём энтропию идеального газа

    S = −(∂F/∂T)_{V,N} = kN{5/2 + ln[(2πmkT)^{3/2}V/(2πℏ)³N]}.         (7.33)

Из формулы (7.33) хорошо видна аддитивность энтропии. Например, если адиабатически смешивать два идеальных газа, имеющих одинаковую температуру, химический состав и концентрацию n и занимающих объёмы V₁ и V₂, то энтропия смеси, занимающей объём V₁ + V₂, будет равна сумме энтропий этих газов до смешивания. Действительно,

    S(N₁ + N₂) = k(nV₁ + nV₂){5/2 + ln[(2πmkT)^{3/2}/n(2πℏ)³]} =
              = Σᵢ₌₁² knVᵢ{5/2 + ln[Vᵢ(2πmkT)^{3/2}/Nᵢ(2πℏ)³]} = S(N₁) + S(N₂).

Для вычисления μ воспользуемся формулой (5.21)

    μ = (∂U/∂N)_{T,V} − T(∂S/∂N)_{T,V}.                                  (7.34)

Подставляя (7.33) и U в (7.34), получим μ = kT ln[N(2πℏ)³/V(2πmkT)^{3/2}].

───────────────────────────────────────────────────────────────────
Пример 7.5. Вычислить теплоёмкость при постоянном объёме идеального одноатомного газа из N частиц, находящегося при температуре T в поле силы тяжести. Газ находится в полубесконечном прямом цилиндре с площадью основания s.
───────────────────────────────────────────────────────────────────

Решение. В цилиндрической системе координат, ось z которой параллельна образующей цилиндра, функция Гамильтона (7.2) принимает вид H = Σᵢ₌₁ᴺ (pᵢ²/2m + U₀(ρᵢ) + mgzᵢ), где U₀(ρᵢ) = 0, если координата ρᵢ ∈ s, и U₀(ρᵢ) = ∞, если ρᵢ ∉ s.

Используя формулу Стирлинга, получаем следующие выражения для статистического интеграла и свободной энергии:

    Z₁(T, V) = (2πmkT)^{3N/2}/[(2πℏ)³ᴺN!] [∫∫ₛ dxᵢdyᵢ ∫₀^∞ exp(−mgzᵢ/kT)dzᵢ]ᴺ =     (7.35)
             = (2πmkT)^{3N/2} sᴺ (kT/mg)ᴺ / [(2πℏ)³ᴺN!],

    F = −kT ln Z₁ = −kTN{1 + ln[(2πmkT)^{3/2}skT/(2πℏ)³mgN]}.          (7.36)

По определению теплоёмкость C̃_V = T(∂S/∂T)_V. Используя формулу S = −(∂F/∂T)_V, получаем, что C̃_V = −T(∂²F/∂T²)_V. Дважды дифференцируя (7.36) и умножая на −T, находим окончательный ответ: C̃_V = 5kN/2. Таким образом, учёт потенциальной энергии гравитационного поля приводит к увеличению теплоёмкости газа.

Пример 7.6. Найти изменение свободной энергии идеального газа, находящегося в центрифуге при температуре T, вызванное её вращением с угловой скоростью ω. Масса одной молекулы m. Потенциальной энергией гравитационного поля пренебречь. Радиус центрифуги R, высота h₀.
───────────────────────────────────────────────────────────────────

Решение. После перехода во вращающуюся с угловой скоростью ω систему отсчёта записанная в цилиндрических координатах функция Гамильтона (7.2) принимает вид H = Σᵢ₌₁ᴺ (pᵢ²/2m + U₀(zᵢ) − mω²ρᵢ²/2), где U₀(zᵢ) = 0, если 0 ≤ zᵢ ≤ h₀, и U₀(zᵢ) = ∞, если zᵢ > h₀.

Используя формулы (7.10), (7.11), получим следующие выражения для статистического интеграла и свободной энергии:

    Z₁(a, T) = (2πmkT)^{3N/2}/[(2πℏ)³ᴺN!] [∫₀^R ρ exp(mω²ρ²/2kT)dρ ∫₀^{2π} dφ ∫₀^{h₀} dz]ᴺ =
             = (2πmkT)^{3N/2}/[(2πℏ)³ᴺN!] [2πh₀kT/(mω²) (exp(mω²R²/2kT) − 1)]ᴺ,

    F = −kTN{1 + ln[(2πmkT)^{3/2}/(2πℏ)³N] + ln[2πh₀kT/(mω²) (exp(mω²R²/2kT) − 1)]}.   (7.37)

При ω → 0 последнее выражение стремится к

    F(ω = 0) = −kTN{1 + ln[(2πmkT)^{3/2}/(2πℏ)³N]} − kTN ln(πh₀R²)

и приращение свободной энергии

    ΔF = F − F(ω = 0) = −kTN ln{(2kT/mω²R²)[exp(mω²R²/2kT) − 1]}.

При больших скоростях вращения центрифуги (ω² >> 2kT/mR²) большинство молекул находятся вдали от оси вращения и

    ΔF ≈ kTN[ln(mω²R²/2kT) − mω²R²/2kT].

───────────────────────────────────────────────────────────────────
Пример 7.7. Найти теплоёмкость и уравнение состояния идеального газа в ультрарелятивистском случае (энергия частицы E связана с импульсом p соотношением E = cp, где c — скорость света).
───────────────────────────────────────────────────────────────────

Решение. Классический статистический интеграл

    Z = ∫exp(−H/kT)dX = (4π ∫₀^∞ p² exp(−cp/kT)dp dr⃗)ᴺ = [8πV(kT/c)³]ᴺ

и, следовательно, свободная энергия F = −kT ln{[(2πℏ)³ᴺN!]⁻¹Z} с учётом формулы Стирлинга имеет вид: F = −kTN{1 + ln[8πV(kT)³/N(2πℏ)³c³]}.

Теплоёмкость C̃_V = −T(∂²F/∂T²)_V = 3Nk = 3νR, т.е. она в два раза превышает теплоёмкость нерелятивистского одноатомного газа. Давление p = −(∂F/∂V)_{T,N} = kTN/V или pV = νRT.
