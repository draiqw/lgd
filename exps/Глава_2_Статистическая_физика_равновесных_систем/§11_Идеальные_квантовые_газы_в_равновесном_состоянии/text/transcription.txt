§11. Идеальные квантовые газы в равновесном состоянии

══════════════════════════════════════════════════════════════════
КРАТКИЕ ТЕОРЕТИЧЕСКИЕ СВЕДЕНИЯ
══════════════════════════════════════════════════════════════════

Квантовое состояние системы, состоящей из N одинаковых невзаимодействующих частиц, определяется набором их квантовых чисел n₁, n₂, ..., nₙ. Здесь n = (n₁, n₁₂, n₁₃, ..., nᵢₛ) — множество, состоящее из s квантовых чисел, определяющих состояние частицы с номером i. Число частиц

    Nₗ = Σᵢ₌₁ᴺ δₗ,ₙᵢ,                                                    (11.1)

таких, что все s квантовых чисел каждой из них совпадают с набором чисел l = (l₁, l₂, ..., lₛ), является случайной функцией, зависящей от состояния системы. В (11.1) δₗ,ₙᵢ = δₗ₁,ₙᵢ₁δₗ₂,ₙᵢ₂...δₗₛ,ₙᵢₛ, где δₗᵢ,ₙᵢⱼ — символ Кронекера. Из условия постоянства числа частиц следует равенство

    Σₗ Nₗ = N.                                                           (11.2)

С помощью введённых обозначений полную энергию Eₙ₁,ₙ₂,...,ₙₛ системы невзаимодействующих частиц можно записать в виде Eₙ₁,ₙ₂,...,ₙₛ = Σₗ EₗNₗ, где Eₗ — энергия частицы, находящейся в квантовом состоянии l.

Если рассматриваемая система частиц находится в термостате при температуре T, то функция распределения её микросостояний Pₙ₁,ₙ₂,...,ₙₛ(N, T) — квантовое каноническое распределение Гиббса (7.8), которое с учётом сказанного выше можно представить в виде

    Pₙ₁,ₙ₂,...,ₙₛ(N, T) = exp(−Eₙ₁,ₙ₂,...,ₙₛ/kT) = exp((Ω + μN − Eₙ₁,ₙ₂,...,ₙₛ)/kT) =    (11.3)
                       = exp((Ω + Σₗ(μ − Eₗ)Nₗ)/kT),

где μ — химический потенциал. В (11.3) большой термодинамический потенциал

    Ω = −kT ln(Σₙ₁,ₙ₂,...,ₙₛ exp[Σₗ(μ − Eₗ)Nₗ/kT]).                        (11.4)

Преобразуя формулы (11.3) и (11.4), легко получить функцию распределения

    fₙₗ = exp{[Ωₗ + (μ − Eₗ)Nₗ]/kT},                                     (11.5)

дающую вероятность того, что в определяемом набором квантовых чисел l₁, l₂, ..., lₛ состоянии находится Nₗ частиц. В (11.5)

    Ωₗ = −kT ln[Σₙₗ exp[(μ − Eₗ)Nₗ/kT]].                                  (11.6)

С помощью (11.5), (11.6) можно найти выражение для среднего числа частиц N̄ₗ, находящихся в квантовом состоянии l: N̄ₗ = −(∂Ωₗ/∂μ)ₜ. Условие (11.2), естественно, выполняется и для средних величин

    Σₗ N̄ₗ = N.                                                          (11.7)

Если квантовые частицы описываются симметричными волновыми функциями (система бозе-частиц, или бозонов), то числа заполнения Nₗ могут быть произвольными: 0 ≤ Nₗ ≤ ∞. В этом случае из (11.6) получается (см. пример 11.1) распределение Бозе

    N̄ₗ = (exp[(Eₗ − μ)/kT] − 1)⁻¹.                                       (11.8)

В случае, когда квантовые частицы описываются антисимметричными волновыми функциями (система ферми-частиц, или фермионов), в соответствии с принципом Паули, Nₗ может принимать только два значения: 0 и 1. Из (11.6) для таких квантовых частиц следует распределение Ферми

    N̄ₗ = (exp[(Eₗ − μ)/kT] + 1)⁻¹.                                       (11.9)

Бозонами являются фотон, π- и K-мезоны. Большинство элементарных частиц (электроны, протоны, нейтроны и др.) являются фермионами.

При больших температурах распределения Бозе и Ферми переходят в распределение Больцмана.

───────────────────────────────────────────────────────────────────
Пример 11.1. Опираясь на каноническое распределение Гиббса (11.3), получить распределение Бозе (11.8).
───────────────────────────────────────────────────────────────────

Решение. Преобразуем выражение (11.4) для входящего в распределение Гиббса большого термодинамического потенциала

    Ω = −kT ln(Σₙ₁,ₙ₂,...,ₙₛ exp[Σₗ(μ − Eₗ)Nₗ/kT]) =                      (11.10)
      = −kT ln(∏ₗ Σₙ₁,ₙ₂,...,ₙₛ exp[(μ − Eₗ)Nₗ/kT] Σₙₗ δₙₗ,αₗ).

В правую часть последнего равенства был включён единичный множитель: Σₙₗ δₙₗ,αₗ, где αₗ = Σᵢ₌₁ᴺ δₗ,ₙᵢ. Далее поменяем в (11.10) порядок суммирования

    Ω = −kT ln(∏ₗ Σₙₗ exp[(μ − Eₗ)Nₗ/kT] Σₙ₁,ₙ₂,...,ₙₛ δₙₗ,αₗ)            (11.11)

и учтём, что входящая в (11.11) достаточно сложная сумма по разным квантовым состояниям n₁, n₂, ..., nₙ при фиксированном значении Nₗ равна единице. Действительно, различные слагаемые в ней отличаются лишь номерами частиц, а в силу их тождественности, состояния, отличающиеся только перестановкой квантовых частиц, являются одинаковыми.

С учётом этого формула (11.11) может быть записана в виде:

    Ω = Σₗ Ωₗ = −kT Σₗ ln Zₗ = −kT Σₗ ln[Σₙₗ exp[(μ − Eₗ)Nₗ/kT]].         (11.12)

Подставляя (11.12) в (11.3), получим, что Pₙ₁,ₙ₂,...,ₙₛ(N, T) = ∏ₗ Pₙₗ, где

    Pₙₗ = exp{[Ωₗ + (μ − Eₗ)Nₗ]/kT} = Zₗ⁻¹ exp[(μ − Eₗ)Nₗ/kT].           (11.13)

Из последней формулы видно, что N̄ₗ = Σₙₗ NₗPₙₗ = −(∂Ωₗ/∂μ)ₜ.

По условию задачи рассматриваемая система частиц подчиняется статистике Бозе, и, следовательно, числа заполнения Nₗ могут принимать любые значения. Поэтому Zₗ — сумма геометрической прогрессии и

    Ωₗ = kT ln{1 − exp[(μ − Eₗ)/kT]}.

Продифференцировав Ωₗ по μ при постоянной температуре, получим N̄ₗ = (exp[(Eₗ − μ)/kT] − 1)⁻¹.

───────────────────────────────────────────────────────────────────
Пример 11.2. Оценить, при какой температуре T распределения Бозе и Ферми переходят в распределение Больцмана.
───────────────────────────────────────────────────────────────────

Решение. Из (11.8), (11.9) видно, что распределения Бозе и Ферми переходят в распределение Больцмана N̄ₗ = exp[(μ − Eₗ)/kT] при выполнении неравенства

    exp[(Eₗ − μ)/kT] >> 1.                                               (11.14)

Если среднее расстояние между частицами достаточно велико, то движение частицы массой m можно описывать законами классической механики, а её внутреннее состояние — набором квантовых чисел n. В отсутствие внешних полей энергия Eₗ = p²/2m + Eₙ, где Eₙ — внутренняя энергия частицы в состоянии n, а p⃗ — её импульс. Подставим Eₗ в N̄ₗ = exp[(μ − Eₗ)/kT], а затем заменим N̄ₗ в (11.7) и заменим в получившемся соотношении суммирование по квантовым числам интегрированием по фазовому пространству, в соответствии с формулой (7.11). В результате получим равенство

    ∫(dr⃗dp⃗/(2πℏ)³) Σₙ exp[(μ − p²/2m − Eₙ)/kT] = N,                     (11.15)

из которого найдём химический потенциал

    μ = kT ln{(N/V)(2πℏ)³/[(2πmkT)^{3/2}][Σₙ exp(−Eₙ/kT)]⁻¹}.            (11.16)

Подставляя μ в (11.14), получим неравенство

    (N/V)(2πℏ)³/(2πmkT)^{3/2} << [Σₙ exp(−Eₙ/kT)][exp(p²/2m + Eₙ)/kT)],

правая часть которого близка к единице даже тогда, когда большая часть атомов находится в основном состоянии. В результате приходим к неравенству N(2πℏ)³ << V(2πmkT)^{3/2}, или T >> T* = N^{2/3}2πℏ²/V^{2/3}mk, при выполнении которого распределения Бозе и Ферми переходят в распределение Больцмана. Температуру T* называют температурой вырождения. Для электронного газа в металлах (m ≈ 10⁻³⁰ кг, N/V ≈ 5·10²⁸ м⁻³) всегда необходимо использовать распределение Ферми, так как неравенство T << T* справедливо вплоть до 10⁴ K (практически для всех температур, при которых могут существовать металлы).

───────────────────────────────────────────────────────────────────
Пример 11.3. Найти термодинамический потенциал Ω и внутреннюю энергию U газа квантовых бесструктурных частиц, подчиняющихся статистике Бозе. Газ находится в объёме V при температуре T, намного большей температуры вырождения. Масса частицы m.
───────────────────────────────────────────────────────────────────

Решение. Большой термодинамический потенциал рассматриваемого газа можно найти, подставив в (11.12) найденную в примере 11.1 формулу для Ωₗ. В полученном выражении

    Ω = Σₗ Ωₗ = kT Σₗ ln{1 − exp[(μ − Eₗ)/kT]}                           (11.17)

энергия Eₗ равна кинетической энергии частицы p²/2m. Заменив в (11.17) суммирование по квантовым числам интегрированием по фазовому пространству в соответствии с формулой (7.11). В результате (11.17) примет вид

    Ω = kT ∫(Vdp⃗/(2πℏ)³) ln{1 − exp[(μ − p²/2m)/kT]}.                    (11.18)

После перехода к сферическим координатам, интегрирования по полярному и азимутальному углам и замене переменной p на (2mE)^{1/2} равенство (11.18) запишется в виде

    Ω = VkTm^{3/2}/(2^{1/2}π²ℏ³) ∫₀^∞ ln{1 − exp[(μ − E)/kT]}E^{1/2}dE.   (11.19)

Интегрируя его по частям, окончательно находим

    Ω = −2^{1/2}Vm^{3/2}/(3π²ℏ³) ∫₀^∞ {exp[(E − μ)/kT] − 1}⁻¹E^{3/2}dE.   (11.20)

В §5 было показано, что внутренняя энергия

    U = Ω − T(∂Ω/∂T)_μ − μ(∂Ω/∂μ)_T.                                    (11.21)

Подставляя (11.20) в (11.21), получим следующее выражение для U:

    U = Vm^{3/2}/(2^{1/2}π²ℏ³) ∫₀^∞ {exp[(E − μ)/kT] − 1}⁻¹E^{3/2}dE.     (11.22)

Знание термодинамических потенциалов Ω и U позволяет также найти энтропию, давление, среднее число частиц, теплоёмкости и другие характеризующие газ макроскопические параметры (см. §4 и §5).

───────────────────────────────────────────────────────────────────
Пример 11.4. Найти выражения для внутренней энергии U, давления p′ и среднего числа частиц N̄ электронного газа, находящегося в объёме V при температуре T. Масса электрона m.
───────────────────────────────────────────────────────────────────

Решение. Электронный газ подчиняется статистике Ферми и, в соответствии с принципом Паули, Nₗ может принимать только два значения: 0 и 1. В этом случае формула (11.6) запишется в виде

    Ωₗ = −kT ln{1 + exp[(μ − Eₗ)/kT]}.                                   (11.23)

Большой термодинамический потенциал рассматриваемого электронного газа можно найти, подставив (11.23) в (11.12) и учтя, что внутреннее состояние электрона характеризуется квантовым числом (спином), принимающим только два значения. Поэтому одну и те же энергию Eₗ могут иметь две частицы. С учётом этого (11.13) запишется в виде

    Ω = Σₗ Ωₗ = −2kT Σₗ ln{1 + exp[(μ − Eₗ)/kT]}.                        (11.24)

Далее надо подставить в Ω энергию Eₗ, которая равна кинетической энергии частицы p²/2m, и заменить суммирование по квантовым числам интегрированием по фазовому пространству в соответствии с формулой (7.11). В результате (11.24) примет вид

    Ω = −2kT ∫(Vdp⃗/(2πℏ)³) ln{1 + exp[(μ − p²/2m)/kT]}.                  (11.25)

После перехода к сферическим координатам, интегрирования по полярному и азимутальному углам и замене переменной p на (2mE)^{1/2} равенство (11.25) запишется в виде

    Ω = −2^{1/2}VkTm^{3/2}/(π²ℏ³) ∫₀^∞ ln{1 + exp[(μ − E)/kT]}E^{1/2}dE.   (11.26)

Интегрируя его по частям, окончательно находим:

    Ω = −2^{3/2}Vm^{3/2}/(3π²ℏ³) ∫₀^∞ {exp[(E − μ)/kT] + 1}⁻¹E^{3/2}dE.   (11.27)

Пользуясь формулой (11.21), получим выражение для внутренней энергии

    U = 2^{1/2}Vm^{3/2}/(π²ℏ³) ∫₀^∞ {exp[(E − μ)/kT] + 1}⁻¹E^{3/2}dE.     (11.28)

Давление идеального газа p′ связано с U соотношением p′ = 2U/3V.

Среднее число частиц найдём, дифференцируя термодинамический потенциал Ω:

    N̄ = −∂Ω/∂μ = 2^{1/2}Vm^{3/2}/(π²ℏ³) ∫₀^∞ {exp[(E − μ)/kT] + 1}⁻¹E^{1/2}dE.   (11.29)

В полученные выражения для внутренней энергии, давления и среднего числа частиц входит множитель {exp[(E − μ)/kT] + 1}⁻¹. При T → 0 он стремится к единице, если E ≤ μ, и к нулю, если E > μ, поэтому выражение (11.29) принимает вид

    N̄ = 2^{1/2}Vm^{3/2}/(π²ℏ³) ∫₀^μ E^{1/2}dE = 2^{3/2}Vm^{3/2}μ^{3/2}/(3π²ℏ³).

Максимально возможная при нулевой температуре энергия электронов Eₖ = (3π²N̄/V)^{2/3}ℏ²/2m называется энергией Ферми. При абсолютном нуле температуры заполнены все нижние энергетические уровни, вплоть до энергии Ферми. Внутренняя энергия (11.28) в этом случае выражается через энергию Ферми: U = 3N̄Eₖ/5, а уравнение состояния принимает вид p′ = 2U/3V = (3π²)^{2/3}(N/V)^{5/3}ℏ²/5m.

───────────────────────────────────────────────────────────────────
Пример 11.5. Найти энергию колебательного движения молекулы идеального двухатомного газа при температуре T, считая каждую молекулу линейным квантовым гармоническим осциллятором, имеющим бесконечный набор энергетических уровней Eₙ = (n + 1/2)ℏω, где n = 0, 1, 2, ...
───────────────────────────────────────────────────────────────────

Решение. Найдём статистическую сумму (7.9) одного осциллятора

    Z₁ = Σₙ₌₀^∞ exp(−Eₙ/kT) = exp(−ℏω/2kT) Σₙ₌₀^∞ exp(−nℏω/kT) =         (11.30)
       = exp(−ℏω/2kT)[1 − exp(−ℏω/kT)]⁻¹

и вычислим с её помощью связанную с колебательным движением внутреннюю энергию

    U = F − T(∂F/∂T)_V = −kT ln Z₁ + kT ln Z₁ + kT² ∂ln(Z₁)/∂T =          (11.31)
      = ℏω/2 + ℏω[exp(ℏω/kT) − 1]⁻¹.

При T = 0 U имеет конечное значение ℏω/2, а при T >> ℏω/k — U ≈ kT.

───────────────────────────────────────────────────────────────────
Пример 11.6. Найти выражение для теплоёмкости C̃_V кристалла из N атомов, считая, что гармонические колебания атомов в среде можно заменить 3N квантовыми осцилляторами, каждый из которых имеет бесконечный набор энергетических уровней Eₙ = (n + 1/2)ℏω, где n = 0, 1, 2, ...
───────────────────────────────────────────────────────────────────

Решение. Пользуясь формулой (11.31) легко получить внутреннюю энергию системы, состоящей из 3N осцилляторов

    U = 3Nℏω/2 + 3Nℏω[exp(ℏω/kT) − 1]⁻¹.

Теплоёмкость твёрдого тела при постоянном объёме выражается формулой

    C̃_V = (∂U/∂T)_V = 3Nk(ℏω/kT)² exp(ℏω/kT){exp(ℏω/kT) − 1}⁻².        (11.32)

Зависимость C̃_V от T, задаваемая равенством (11.31), качественно согласуется с экспериментом.

Действительно, если ℏω/kT << 1, то C̃_V ≈ 3Nk. При стремлении температуры к нулю

    C̃_V → 3Nk(ℏω/kT)² exp(−ℏω/kT) → 0.

Однако эксперимент даёт лишь степенное уменьшение C̃_V ~ T³ при малых T, поэтому при низких температурах используемая модель нуждается в уточнении.
