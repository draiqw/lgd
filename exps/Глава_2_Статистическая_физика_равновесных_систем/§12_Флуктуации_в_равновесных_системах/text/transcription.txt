§12. Флуктуации в равновесных системах

═══════════════════════════════════════════════════════════════════
КРАТКИЕ ТЕОРЕТИЧЕСКИЕ СВЕДЕНИЯ
═══════════════════════════════════════════════════════════════════

Хаотическое движение частиц может привести к флуктуациям — самопроизвольным отклонениям функций динамических переменных B_j(X) от их средних (равновесных) значений B̄_j, где j = 1, 2, ..., n. Здесь, как и ранее, X = (x₁, x₂, ..., x_N) = (r̄₁, r̄₂, ..., r̄_N, p̄₁, p̄₂, ..., p̄_N) — совокупность значений координат и компонент импульсов всех N частиц системы, черта означает статистическое усреднение.

Флуктуации B_j достаточно полно характеризуются дисперсией (квадратичной флуктуацией) этой величины:

(B_j - B̄_j)² = (ΔB_j)² = B_j² - (B̄_j)² = σ²_{B_j}

и относительной флуктуацией δ_{B_j} = σ_{B_j}/B̄_j.

Связь между функциями B_i и B_j характеризуется коэффициентом корреляции K_{ij} = ΔB_i·ΔB_j.

───────────────────────────────────────────────────────────────────
Распределения Гиббса и термодинамические параметры
───────────────────────────────────────────────────────────────────

Микроканоническое, каноническое и большое каноническое распределения Гиббса соответствуют различным наборам внешних термодинамических параметров a₁, a₂, ..., a_n (см. §1), которые задают систему, точно известны и не флуктуируют. Сопряжённые им внутренние термодинамические параметры B₁, B₂, ..., B_n испытывают флуктуации.

Характеристики последних могут быть легко вычислены, если удастся найти плотность функции распределения w(B_j), связанную с микроканоническим, каноническим или большим каноническим распределениями Гиббса соотношением:

w(B_j) = ∫δ(B_j - B_j(X))w_N(X, a)dX

В частности, если w(B_j) оказывается распределением Гаусса:

w(B_j) = (λ/2π)^(1/2) exp(-λB_j²/2),                    (12.1)

то дисперсия B_j задаётся формулой (ΔB_j)² = 1/λ.

───────────────────────────────────────────────────────────────────
Квазитермодинамическая теория флуктуаций
───────────────────────────────────────────────────────────────────

Вычисление флуктуаций с помощью равновесных функций распределения представляет собой задачу, сравнимую по сложности с вычислением средних значений термодинамических потенциалов. Поэтому часто используется приближённая квазитермодинамическая теория, в которой считается, что малая часть термодинамической системы также может характеризоваться термодинамическими параметрами.

При этом малые флуктуации могут рассматриваться как переход системы из первоначально равновесного состояния, характеризующегося параметрами a₁, a₂, ..., a_n, в новое квазиравновесное состояние, определяемое наряду с a₁, a₂, ..., a_n дополнительными внешними параметрами. В этом случае последним соответствуют фиктивные термодинамические силы, «удерживающие» термодинамическую систему в новом равновесном состоянии.

Этот переход считается очень медленным. Иными словами, все рассматриваемые временные интервалы значительно превышают время релаксации к состоянию равновесия, в том числе и для областей, в которых произошла флуктуация. Кроме того, флуктуации в различных микрообластях системы считаются независимыми. Последнее возможно, если они малы и происходят достаточно быстро по сравнению со скоростью релаксации всей системы.

По А. Эйнштейну вероятность w такого перехода из первоначально равновесного состояния в новое квазиравновесное состояние задаётся формулой:

w = exp(ΔS'/k), где ΔS' = ΔS + ΔS_T

— происходящее при этом изменение энтропии, которое складывается из изменений энтропии рассматриваемой системы S и термостата S_T, в равновесии с которым она находится, k — постоянная Больцмана.

Учитывая неизменность суммарной энергии, полного объёма и общего числа частиц в рассматриваемой системе и термостате, а также условия равновесия (5.15), (5.16) между ними, вероятность перехода в новое квазиравновесное состояние можно записать в виде:

w = C exp[-(ΔU + pΔV - μΔN - TΔS)/kT],                  (12.2)

где C — определяемая из условия нормировки константа, U — внутренняя энергия, p — давление, V — объём, μ — химический потенциал, T — температура. Показатель экспоненты в (12.2) пропорционален работе вышеупомянутых фиктивных термодинамических сил, «удерживающих» термодинамическую систему в состоянии квазиравновесия.

───────────────────────────────────────────────────────────────────
Частные случаи формулы (12.2)
───────────────────────────────────────────────────────────────────

Для изолированной системы (постоянные U, V, N), системы в термостате (постоянные T, V, N) и системы в термостате с переменным числом частиц (постоянные T, V, μ) формула (12.2) переходит соответственно в выражения:

w = C exp(ΔS/k),                                        (12.3)

w = C exp[-(ΔU - TΔS)/kT] = C exp(-ΔF/kT),              (12.4)

w = C exp[-(ΔU - μΔN - TΔS)/kT] = C exp(-ΔΩ/kT),        (12.5)

где F(T, V, N) — свободная энергия, Ω(T, V, μ) — большой термодинамический потенциал.

Разлагая с точностью до квадратичных слагаемых U(S, V, N) в ряд Тейлора и учитывая, что (∂U/∂S)_{V,N} = T, (∂U/∂V)_{T,N} = -p, (∂U/∂N)_{V,T} = μ, формулу (12.2) можно записать в виде:

w = C exp[(ΔpΔV - ΔμΔN - ΔTΔS)/2kT].                    (12.6)

Равенство (12.6) часто называют общей формулой для вероятности флуктуаций состояния неизолированной системы.

───────────────────────────────────────────────────────────────────
Пример 12.1
───────────────────────────────────────────────────────────────────

Для системы с переменным числом частиц доказать равенство:

kT²(∂U/∂T)_{V,N} = H̄² - Ū²,

где внутренняя энергия U является средним значением гамильтониана H(X).

Решение. Продифференцируем по T при постоянных V и N внутреннюю энергию системы:

U = ∫H(X)exp{[Ω + μN - H(X)]/kT}dX

и умножим (∂U/∂T)_{V,N} на kT². В итоге получим:

kT²(∂U/∂T)_{V,N} = kT² ∫H(X)exp[(Ω + μN - H(X))/kT] × [1/kT(∂Ω/∂T)_{V,μ} - (Ω + μN - H(X))/kT²]dX     (12.7)
                 = U[T(∂Ω/∂T)_{V,μ} - (Ω + μN)] + H̄²(X).

При записи последнего равенства учтено, что U = H̄(X).

После подстановки в (12.7) (∂Ω/∂T)_V = -S (см. формулу (5.12)) и учёта равенства между термодинамическими потенциалами Ω = U - TS - μN, окончательно находим:

kT²(∂U/∂T)_V = H̄²(X) - U(T̄S + Ω + μN) = H̄²(X) - U².

───────────────────────────────────────────────────────────────────
Пример 12.2
───────────────────────────────────────────────────────────────────

Используя большое каноническое распределение Гиббса, доказать равенство:

(ΔN)² = N̄² - (N̄)² = kT(∂N̄/∂μ)_{T,V}.

Решение. Среднее число частиц N̄ и среднее значение квадрата числа частиц N̄² в термодинамической системе, имеющей постоянный объём V и температуру T, выражаются с помощью формул (7.17), (7.18):

N̄ = Z̃₂⁻¹ Σ_N (N/(2πℏ)^{3N} N!) ∫exp[(μN - H(X, N, a))/kT]dX = kT/Z̃₂ · ∂Z̃₂/∂μ,     (12.8)

N̄² = Z̃₂⁻¹ Σ_N (N²/(2πℏ)^{3N} N!) ∫exp[(μN - H(X, N, a))/kT]dX = (kT)²/Z̃₂ · ∂²Z̃₂/∂μ²,     (12.9)

где Z̃₂ = Σ_N ((2πℏ)^{3N} N!)⁻¹ ∫exp[(μN - H(X, N, a))/kT]dX.

Продифференцируем соотношение (7.16) при постоянных T и V по μ, умножим получившееся выражение на kT и учтём, что большой термодинамический потенциал Ω(V, T, μ) = -kT ln Z̃₂. В итоге получим равенство:

kT(∂N̄/∂μ)_{T,V} = -kT · ∂²Ω/∂μ² = (kT)² · ∂/∂μ(∂Z̃₂/∂μ / Z̃₂) =
                 = (kT)²/Z̃₂[∂²Z̃₂/∂μ² - 1/Z̃₂(∂Z̃₂/∂μ)²] = N̄² - (N̄)² = (ΔN)²,     (12.10)

которое необходимо было доказать.

───────────────────────────────────────────────────────────────────
Пример 12.3
───────────────────────────────────────────────────────────────────

Выразить коэффициент корреляции между флуктуациями энергии и числа частиц для системы, описываемой большим каноническим распределением Гиббса, через уравнение состояния и дисперсию числа частиц.

Решение. Перепишем равенство (12.8) с помощью большого термодинамического потенциала Ω(V, T, μ) = -kT ln Z̃₂ и получившееся выражение продифференцируем по T при постоянных V и μ:

(∂N̄/∂T)_{V,μ} = Σ_N (N/(2πℏ)^{3N} N!) ∫exp[(Ω + μN - H)/kT] × [1/kT(∂Ω/∂T)_{V,μ} - (Ω + μN - H)/kT²]dX.

Учитывая, что (∂Ω/∂T)_{V,μ} = -S (см. формулу (5.12)), а Ω + TS + μN̄ = U, формулу для (∂N̄/∂T)_{V,μ} можно переписать в виде:

(∂N̄/∂T)_{V,μ} = [-ŪN̄ + μ(N̄)² - μN̄² + H̄N]/kT².                    (12.11)

Проверим коэффициент корреляции ΔNΔH с помощью (12.11) и (12.10):

ΔNΔH = (N - N̄)(H - U) = N̄H - ŪN̄ = kT²(∂N̄/∂T)_{V,μ} + μ(N̄² - (N̄)²) = kT²(∂N̄/∂T)_{V,μ} + kTμ(∂N̄/∂μ)_{T,V}.     (12.12)

Поделим теперь очевидное равенство:

dN̄(T, V, μ) = (∂N̄/∂T)_{V,μ}dT + (∂N̄/∂V)_{T,μ}dV + (∂N̄/∂μ)_{V,T}dμ

на dμ и зафиксируем V и N̄. В результате получим соотношение:

0 = (∂N̄/∂T)_{V,μ}(∂T/∂μ)_{V,N̄} + (∂N̄/∂μ)_{T,V},

из которого выразим производную (∂N̄/∂T)_{V,μ} и подставим её в формулу (12.12), которая после этого примет вид:

ΔNΔH = kT(∂N̄/∂T)_{T,V}[μ - T(∂μ/∂T)_{V,N̄}].                    (12.13)

Сомножитель перед квадратной скобкой в (12.13) согласно (12.10) равен дисперсии числа частиц. Преобразуем выражение в квадратных скобках, используя формулу (5.6):

μ - T(∂μ/∂T)_{V,N̄} = (∂F/∂N)_{V,T} - T(∂²F/∂N∂T)_{V,T} = ∂/∂N(F - T(∂F/∂T)_{N})_{V,T} = (∂U/∂N)_{V,T}.

При записи последнего равенства мы учли, что (∂F/∂T)_{V,N} = -S, а F + TS = U.

В итоге (12.13) принимает окончательный вид:

ΔNΔH = (ΔN)²(∂U/∂N)_{V,T}.                                (12.13)

Для идеального газа (ΔN)² = N̄, U = 3kTN/2, и, следовательно, ΔNΔH = 3kTN̄/2.

───────────────────────────────────────────────────────────────────
Пример 12.4
───────────────────────────────────────────────────────────────────

В цилиндре под поршнем под давлением p находится N молекул идеального газа при температуре T. Рассчитать дисперсию объёма газа.

Решение. Изменение давления газа равно Δp = (∂p/∂V)_{T,N}ΔV, т.к. ΔT = 0, ΔN = 0. При этом формула (12.6) принимает вид:

w = C exp[(∂p/∂V)_{T,N}(ΔV)²/2kT].

Следовательно, в соответствии с формулой (12.1):

(ΔV)² = -kT(∂V/∂p)_{T,N} = N(kT/p)²/N = V²/N.

───────────────────────────────────────────────────────────────────
Пример 12.5
───────────────────────────────────────────────────────────────────

В цилиндре объёмом V при температуре T находится переменное число частиц N, задаваемое химическим потенциалом μ. Рассчитать дисперсию числа частиц газа.

Решение. По условию ΔV = ΔT = 0, а Δμ = (∂μ/∂N)_{V,T}ΔN. При этом формула (12.6) принимает вид:

w(ΔN) = C exp[-(∂μ/∂N)_{V,T}(ΔN)²/2kT].

Следовательно, в соответствии с формулой (12.1):

(ΔN)² = kT(∂N/∂μ)_{V,T}.

───────────────────────────────────────────────────────────────────
Пример 12.6
───────────────────────────────────────────────────────────────────

Найти дисперсию внутренней энергии U(V, T) идеального одноатомного газа, состоящего из N частиц.

Решение. Усреднив очевидное равенство:

(ΔU)² = (∂U/∂T)_V²(ΔT)² + 2(∂U/∂T)_V(∂U/∂V)_T·ΔTΔV + (∂U/∂V)_T²(ΔV)²,

получаем:

(ΔU)² = (∂U/∂T)_V²(ΔT)² + 2(∂U/∂T)_V(∂U/∂V)_T·ΔTΔV + (∂U/∂V)_T²(ΔV)².

В переменных T и V:

ΔS = (∂S/∂T)_V·ΔT + (∂S/∂V)_T·ΔV,
Δp = (∂p/∂T)_V·ΔT + (∂p/∂V)_T·ΔV.                        (12.14)

С учётом (12.14), постоянства числа частиц и одного из соотношений Максвелла (3.9) формула (12.6) принимает вид:

w = C exp[-(∂S/∂T)_V(ΔT)² - (∂p/∂V)_T(ΔV)²)/(2kT)].     (12.15)

Из (12.15) следует (см. также равенство (12.1)), что:

(ΔT)² = kT(∂T/∂S)_V,
(ΔV)² = -kT(∂V/∂p)_T и ΔTΔV = 0.

Подставляя (ΔT)² и (ΔV)² в выражение для (ΔU)², получаем:

(ΔU)² = kT[(∂U/∂T)_V²(∂T/∂S)_{V,N} - (∂p/∂V)_{T,N}(∂U/∂V)_T²].     (12.16)

Внутренняя энергия одноатомного идеального газа U = 3NkT/2, а его энтропия S = kN{5/2 + ln[(2πmkT)^(3/2)V/(2πℏ)³N]} (см. формулу 7.33), и поэтому (12.16) принимает вид:

(ΔU)² = 3N(kT)²/2.

Заметим, что последняя формула также непосредственно следует из ответа примера 12.1 в случае одноатомного идеального газа.
