Глава 3. ЭЛЕМЕНТЫ ТЕОРИИ НЕРАВНОВЕСНЫХ ПРОЦЕССОВ

§13. Кинетическое уравнение Больцмана и процессы переноса

═══════════════════════════════════════════════════════════════════
КРАТКИЕ ТЕОРЕТИЧЕСКИЕ СВЕДЕНИЯ
═══════════════════════════════════════════════════════════════════

При переходе из неравновесного состояния в равновесное временна́я эволюция плотности распределения вероятностей микросостояний w_N(x₁, x₂, ..., x_N, t) термодинамической системы, состоящей из N (N ~ N_A) классических одинаковых бесструктурных частиц массой m, имеющих координаты и компоненты импульсов x₁, x₂, ..., x_N, где xᵢ = (r̄ᵢ, p̄ᵢ), описывается уравнением Лиувилля (7.3). Его решение для больших значений N, естественно, невозможно. Однако, в этом и нет необходимости, т.к. для вычисления большинства термодинамических и газодинамических функций достаточно знать функции распределения w_S(x₁, x₂, ..., x_S, t), где S = 1, 2, 3, ..., связанные с w_N(x₁, x₂, ..., x_N, t) соотношением:

w_S(x₁, x₂, ..., x_S, t) = V^S ∫w_N(x₁, x₂, ..., x_N, t)dx_{S+1}dx_{S+2}...dx_N     (13.1)

и удовлетворяющие условию нормировки:

V^{-S} ∫w_S(x₁, x₂, ..., x_S, t)dx₁dx₂...dx_S = 1.     (13.2)

Здесь V — занимаемый газом объём. Так, например, с помощью одночастичной функции распределения находятся средняя скорость упорядоченного движения ū(r̄, t) = NV⁻¹∫(p̄/m)w₁(r̄, p̄, t)dp̄ и плотность числа частиц n(r̄, t) = NV⁻¹∫w₁(r̄, p̄, t)dp̄, а с помощью одночастичной и двухчастичной функций распределения — внутренняя энергия:

U(t) = (N/V)∫(p̄²/2m)w₁(r̄, p̄, t)dp̄dr̄ + (N²/2V²)∫Φ(|r̄₁ - r̄₂|)w₂(r̄₁, p̄₁, r̄₂, p̄₂, t)dp̄₁dr̄₁dp̄₂dr̄₂.     (13.3)

───────────────────────────────────────────────────────────────────
Цепочка уравнений ББГКИ
───────────────────────────────────────────────────────────────────

Здесь Φ(|r̄₁ - r̄₂|) — потенциальная энергия взаимодействия двух частиц. Для нахождения временно́й эволюции этих и других величин необходимо знать уравнения для неравновесных функций распределения w₁(x₁, t), w₂(x₁, x₂, t), и т.д., которые можно получить из уравнения Лиувилля. Математическим следствием (7.3) является бесконечная цепочка уравнений Боголюбова — Борна — Грина — Кирквуда — Ивона (ББГКИ):

(∂/∂t + p̄₁/m · ∂/∂r̄₁ - ∂U₀(r̄₁)/∂r̄₁ · ∂/∂p̄₁)w₁(r̄₁, p̄₁, t) =
= (N/V)∫[∂Φ(|r̄₁ - r̄₂|)/∂r̄₁]∂w₂(r̄₁, p̄₁, r̄₂, p̄₂, t)/∂p̄₁ dr̄₂dp̄₂,     (13.4)

(∂/∂t + p̄₁/m · ∂/∂r̄₁ + p̄₂/m · ∂/∂r̄₂ - ∂U₀(r̄₁)/∂r̄₁ · ∂/∂p̄₁ - ∂U₀(r̄₂)/∂r̄₂ · ∂/∂p̄₂
- ∂Φ(|r̄₁ - r̄₂|)/∂r̄₁ · ∂/∂p̄₁ - ∂Φ(|r̄₁ - r̄₂|)/∂r̄₂ · ∂/∂p̄₂)w₂(r̄₁, p̄₁, r̄₂, p̄₂, t) =
= (N/V)∫[∂Φ(|r̄₁ - r̄₃|)/∂r̄₁ · ∂/∂p̄₁ + ∂Φ(|r̄₂ - r̄₃|)/∂r̄₂ · ∂/∂p̄₂]w₃(r̄₁, p̄₁, r̄₂, p̄₂, r̄₃, p̄₃, t)dr̄₃dp̄₃,     (13.5)

.........

Решение этой цепочки уравнений столь же сложно, как и решение исходного уравнения Лиувилля. Однако, наличие малых параметров и возможность использования различных приближений позволяет оборвать цепочку уравнений и свести задачу к решению замкнутой системы уравнений, например, для функций w₁ и w₂.

───────────────────────────────────────────────────────────────────
Корреляционные функции
───────────────────────────────────────────────────────────────────

Вместо функций w₁, w₂, w₃, ... иногда используют w₁ и последовательность корреляционных функций:

g₂(x₁, x₂, t) = w₂(x₁, x₂, t) - w₁(x₁, t)w₁(x₂, t),     (13.6)

g₃(x₁, x₂, x₃, t) = w₃(x₁, x₂, x₃, t) - w₁(x₁, t)w₁(x₂, t)w₁(x₃, t)
- w₁(x₁, t)g₂(x₂, x₃, t) - w₁(x₂, t)g₂(x₁, x₃, t) - w₁(x₃, t)g₂(x₁, x₂, t),     (13.7)

характеризующих статистическую связь между частицами. Для разреженного газа g₃ ≈ 0, а g₂ ≠ 0 лишь при сближении частиц на расстояние порядка r₀ (см. рис. 4.1), т.е. при их столкновении. Правая часть уравнения (13.4) зависит от двухчастичной корреляционной функции g₂ и поэтому называется интегралом столкновений. Его явный вид определяется моделью рассматриваемого газа и сделанными приближениями.

───────────────────────────────────────────────────────────────────
Кинетическое уравнение Больцмана
───────────────────────────────────────────────────────────────────

Например, для разреженного газа (Nr₀³ << V) в приближении парных абсолютно упругих столкновений и полного ослабления начальных корреляций (рассматриваемый временной интервал изменения w₁ намного больше времени корреляции τ₁ = r₀/v̄, но намного меньше времени релаксации τ₂ = l/v̄, где l — длина свободного пробега, v̄ — характерная скорость теплового движения частиц) уравнение (13.4) принимает вид:

(∂/∂t + p̄₁/m · ∂/∂r̄₁ - ∂U₀(r̄₁)/∂p̄₁ · ∂/∂p̄₁)w₁(r̄₁, p̄₁, t) = (N/mV)∫₀^∞ dp₂ρdρ ×
× ∫₀^{2π} dφ|p̄₂ - p̄₁|[w₁(r̄₁, p̄₂', t)w₁(r̄₁, p̄₁', t) - w₁(r̄₁, p̄₂, t)w₁(r̄₁, p̄₁, t)] ≡ I.     (13.8)

Здесь p̄₁ и p̄₂ — импульсы первой и второй частиц до столкновения, а p̄₁' и p̄₂' — после, ось 0z цилиндрической системы координат ρ, φ, z направлена вдоль вектора p̄₂ - p̄₁. Входящие в (13.8) импульсы p̄₁' и p̄₂' однозначно выражаются через p̄₁ и p̄₂ из соотношений, следующих из законов сохранения импульса и энергии для точечных частиц. Уравнение (13.8) называют кинетическим уравнением Больцмана с интегралом столкновений в форме Н.Н. Боголюбова. Для его решения используются, как правило, различные приближённые методы.

───────────────────────────────────────────────────────────────────
Равновесное распределение Максвелла-Больцмана
───────────────────────────────────────────────────────────────────

В равновесном случае распределение Максвелла-Больцмана:

w₁^{(0)} = 1/(2πmkT)^{3/2} exp(-(p² + 2mU₀(r̄))/kT)(∫exp(-U₀(r̄)/kT)dr̄)⁻¹     (13.9)

является решением (13.8). В этом легко убедиться, подставляя (13.9) в (13.8) и учтя соотношения, следующие из законов сохранения импульса и энергии.

───────────────────────────────────────────────────────────────────
Релаксационное приближение
───────────────────────────────────────────────────────────────────

С помощью уравнения Больцмана можно получить систему уравнений, описывающую процессы переноса в необходимом для описания процессов переноса приближении по газодинамическому параметру l/L, где l — длина свободного пробега, L — характерный размер термодинамической системы.

Например, при приближении термодинамической системы к состоянию равновесия временну́ю эволюцию w₁(r̄, p̄, t) удобно приближённо описывать феноменологическим линейным уравнением с релаксационным членом:

(∂/∂t + p̄/m · ∂/∂r̄ - ∂U₀(r̄)/∂r̄ · ∂/∂p̄)w₁(r̄, p̄, t) = -[w₁(r̄, p̄, t) - w₁^{(0)}(r̄, p̄)]/τ.     (13.10)

Здесь τ — характерное время, в течение которого система достигает равновесного состояния. В ряде случаев τ порядка среднего времени между двумя последовательными соударениями молекулы.

───────────────────────────────────────────────────────────────────
Процессы переноса
───────────────────────────────────────────────────────────────────

С помощью (13.8) (или (13.10)) можно получить уравнения, описывающие в газах процессы переноса — необратимые процессы, в результате которых в физической системе происходит пространственное перераспределение вещества, импульса, энергии или какой-либо другой физической величины. Наиболее важными из них являются диффузия, вязкость и теплопроводность.

───────────────────────────────────────────────────────────────────
Диффузия
───────────────────────────────────────────────────────────────────

Если в некоторой части термодинамической системы плотность числа частиц n(r̄, t) отклонилась от равновесного значения, то в системе возникает движение частиц в таких направлениях, чтобы плотность стала одинаковой во всём объёме. Связанный с этим движением перенос вещества называется диффузией (в случае одинаковых частиц — самодиффузией).

Если концентрация одинаковых молекул может меняться только вдоль оси 0x, то поток частиц u(x, t)n(x, t) (среднее число частиц, пересекающих в единицу времени поверхность единичной площади, перпендикулярную оси 0x) вдоль неё пропорционален производной от концентрации:

u(x, t)n(x, t) = -D∂n/∂x,     (13.11)

где D — коэффициент диффузии. Грубая оценка показывает, что D порядка v̄l. Уравнение баланса числа частиц между двумя параллельными плоскостями приводит в случае ∂D/∂x = 0 к уравнению диффузии:

∂n/∂t = D∂²n/∂x².     (13.12)

Его решение для бесконечной среды с начальным условием n(x, 0) = ñ(x) имеет вид:

n(x, t) = (4πDt)^{-1/2} ∫_{-∞}^{∞} ñ(x')exp[-(x - x')²/4Dt]dx'.     (13.13)

───────────────────────────────────────────────────────────────────
Вязкость
───────────────────────────────────────────────────────────────────

При относительном движении различных частей газа могут возникнуть силы торможения, связанные с вызванными тепловым движением процессами переноса импульса упорядоченного движения между слоями. Возникающее в результате этого уменьшение скорости относительного движения между слоями называется вязкостью.

Если некоторая среда находится между двумя параллельными плоскостями, одна из которых (x = 0) покоится, а другая (x = a) движется параллельно оси 0y со скоростью ū₀ (см. рис. 13.1), то её частицы, передвигаясь вследствие теплового движения вдоль оси 0x, также вовлекаются в упорядоченное движение вдоль оси 0y со скоростью v_y(x).

Изменение v_y с ростом x приводит к появлению направленной противоположно ū₀ силы, величина которой пропорциональна площади S. Поверхностная плотность f_y модуля этой силы удовлетворяет уравнению:

f_y = -η(∂v_y/∂x),     (13.14)

где η — коэффициент вязкости. Грубая оценка показывает, что η порядка v̄/ρ, где ρ — средняя плотность среды.

───────────────────────────────────────────────────────────────────
Теплопроводность
───────────────────────────────────────────────────────────────────

При отклонении температуры в некоторой ограниченной области термодинамической системы от равновесного значения возникает передача теплоты в таких направлениях, чтобы температура во всех её частях стала одинаковой. Связанный с этим процесс переноса теплоты называется теплопроводностью. Подчеркнем, что движение частиц в процессах переноса связано с переходом системы от менее вероятного к более вероятному состоянию.

Если температура газа T(x, t) может меняться только вдоль оси 0x, то проекция вектора плотности теплового потока на ось 0x (среднее количество теплоты, переносимое в единицу времени через единичную плоскую площадку, перпендикулярную оси 0x) пропорциональна производной от температуры:

J_x = -κ∂T/∂x,     (13.15)

где κ — коэффициент теплопроводности. Для него имеет место грубая оценка κ ≈ v̄/ρc_v, где c_v — удельная теплоёмкость газа при постоянном объёме. Заметим, что J_x выражается через одночастичную функцию распределения:

J_x = (N/V)∫(p_x/m - u_x)[(p̄ - mū)²/2m]w₁(r̄, p̄, t)dp̄.

В общем случае записанное для произвольного объёма уравнение баланса энергии приводит к уравнению теплопроводности:

c_vρ∂T/∂t = κΔT + q.     (13.16)

Здесь Δ — оператор Лапласа, q(r̄, t) — плотность мощности тепловых источников (количество теплоты, выделяемой или поглощаемой в единице объёма за единицу времени благодаря различным физическим или химическим процессам).

Методы решений уравнений (13.12), (13.16) с различными граничными и начальными условиями к кругу вопросов, подробно обсуждаемых в курсе математической физики.

───────────────────────────────────────────────────────────────────
Пример 13.1
───────────────────────────────────────────────────────────────────

Выразить через одночастичную функцию распределения концентрацию частиц, плотность газа, среднюю скорость их упорядоченного движения и локальную температуру. Масса одной частицы m.

Решение. Концентрацию частиц n(r̄, t) найдём, усредняя микроскопическую плотность числа частиц:

n(r̄, t) = Σᵢ₌₁ᴺ δ(r̄ - r̄ᵢ(t)) = ∫Σᵢ₌₁ᴺ δ(r̄ - r̄ᵢ(t))w_N(x₁, ..., x_N, t)dx₁...dx_N =
= V⁻¹ Σᵢ₌₁ᴺ ∫δ(r̄ - r̄ᵢ(t))w₁(xᵢ, t)dxᵢ = NV⁻¹∫δ(r̄ - r̄₁(t))w₁(x₁, t)dx₁ =
= NV⁻¹∫w₁(r̄, p̄, t)dp̄.     (13.17)

Плотность массы ρ(r̄, t) связана с концентрацией соотношением ρ(r̄, t) = mn(r̄, t).

Скорость упорядоченного движения частиц ū(r̄, t) определим, приравняв mū(r̄, t)n(r̄, t) среднему значению микроскопической плотности импульса:

Σᵢ₌₁ᴺ p̄ᵢδ(r̄ - r̄ᵢ(t)) = ∫Σᵢ₌₁ᴺ p̄ᵢδ(r̄ - r̄ᵢ(t))w_N(x₁, ..., x_N, t)dx₁...dx_N =
= V⁻¹ Σᵢ₌₁ᴺ ∫p̄ᵢδ(r̄ - r̄ᵢ(t))w₁(xᵢ, t)dxᵢ =
= NV⁻¹∫p̄₁δ(r̄ - r̄₁(t))w₁(r̄, p̄, t)dp̄.     (13.18)

Локальная температура T(r̄, t) одноатомного газа бесструктурных частиц находится из равенства:

3n(r̄, t)kT(r̄, t)/2 = (m/2)Σᵢ₌₁ᴺ [p̄ᵢ/m - u(r̄, t)]²δ(r̄ - r̄ᵢ(t)) =
= (m/2)∫Σᵢ₌₁ᴺ [p̄ᵢ/m - u(r̄, t)]²δ(r̄ - r̄ᵢ(t))w_N(x₁, ..., x_N, t)dx₁...dx_N =
= (m/2V)Σᵢ₌₁ᴺ ∫[p̄ᵢ/m - u(r̄, t)]²δ(r̄ - r̄ᵢ(t))w₁(xᵢ, t)dxᵢ =
= (mN/2V)∫[p̄₁/m - u(r̄, t)]²δ(r̄ - r̄ᵢ(t))w₁(x₁, t)dx₁ =
= (mN/2V)∫[p̄/m - u(r̄, t)]²w₁(r̄, p̄, t)dp̄.     (13.19)

Функции ū(r̄, t), ρ(r̄, t) и T(r̄, t) входят в уравнения газовой динамики, которые с помощью уравнения Больцмана можно получить в необходимом для описания процессов переноса приближении по газодинамическому параметру l/L, где l — длина свободного пробега, L — характерный размер термодинамической системы.

───────────────────────────────────────────────────────────────────
Пример 13.2
───────────────────────────────────────────────────────────────────

Используя уравнение Лиувилля (7.3), получить первое уравнение цепочки уравнений ББГКИ.

Решение. Умножим уравнение Лиувилля на V и проинтегрируем каждое из получившихся слагаемых по x₂, x₃, ..., x_N. В результате член, содержащий производную по времени, с учётом определения (13.1), запишется в виде ∂w₁(r̄₁, p̄₁, t)/∂t.

При интегрировании N-частичной функции распределения по пространственным координатам, выделим из суммы в явном виде первое слагаемое и преобразуем его с учётом (13.1):

(V/m)∫Σᵢ₌₁ᴺ p̄ᵢ[∂w_N(x₁, x₂, ..., x_N, t)/∂r̄ᵢ]dx₂dx₃...dx_N = (p̄₁/m)∂w₁(r̄₁, p̄₁, t)/∂r̄₁ +
+ (V/m)∫Σᵢ₌₂ᴺ p̄ᵢ[∂w_N(x₁, x₂, ..., x_N, t)/∂r̄ᵢ]dx₂dx₃...dr̄ᵢdp̄ᵢ...dx_N.

Содержащая N - 1 слагаемое сумма в правой части последней формулы равна нулю. В этом легко убедиться, осуществив интегрирование каждого слагаемого по dr̄ᵢ и заметив, что вероятность нахождения частицы вне рамок занимаемого газом объёма равна нулю. В итоге в результате преобразования остаётся только (p̄₁/m)∂w₁(r̄₁, p̄₁, t)/∂r̄₁. При определении вклада членов, содержащих ∂U₀(r̄ᵢ)/∂r̄ᵢ, также выделим из суммы в явном виде первое слагаемое и преобразуем его с учётом (13.1):

-V∫Σᵢ₌₁ᴺ [∂U₀/∂r̄ᵢ][∂w_N(x₁, ..., x_N, t)/∂p̄ᵢ]dx₂dx₃...dx_N = -[∂U₀(r̄₁)/∂r̄₁][∂w₁(r̄₁, p̄₁, t)/∂p̄₁] -
- ∫Σᵢ₌₂ᴺ [∂U₀/∂r̄ᵢ][∂w_N(x₁, ..., x_N, t)/∂p̄ᵢ]dx₂dx₃...dr̄ᵢdp̄ᵢ...dx_N.

Все N - 1 слагаемых в правой части последней формулы равны нулю. Действительно, интегрирование каждого из них по dp̄ᵢ в бесконечных пределах требует подстановки w_N(x₁, ..., x_N, t)|_{p_α = ±∞}, где α = x, y, z. Но вероятность того, что частица имеет бесконечно большую скорость, естественно, равна нулю. В итоге, в результате преобразования остаётся только -[∂U₀(r̄₁)/∂r̄₁][∂w₁(r̄₁, p̄₁, t)/∂p̄₁]. Перенесём слагаемые, учитывающие взаимодействие частиц, в правую часть и осуществим интегрирование:

V∫Σᵢ₌₁ᴺ [∂/∂r̄ᵢ Σⱼ₌₂,ⱼ≠ᵢᴺ Φ(|r̄ᵢ - r̄ⱼ|)][∂w_N(x₁, x₂, ..., x_N, t)/∂p̄ᵢ]dx₂dx₃...dx_N =
= V∫[∂/∂r̄₁ Σⱼ₌₂ᴺ Φ(|r̄₁ - r̄ⱼ|)][∂w_N(x₁, x₂, ..., x_N, t)/∂p̄₁]dx₂dx₃...dx_N +
+ V∫Σₘ₌₂ᴺ [∂/∂r̄ₘ Σⱼ₌₁,ⱼ≠ₘᴺ Φ(|r̄₂ - r̄ⱼ|)][∂w_N(x₁, x₂, ..., x_N, t)/∂p̄ₘ]dx₂dx₃...dp̄ₘdr̄ₘ...dx_N =
= V⁻¹∫[∂/∂r̄₁ Σⱼ₌₂ᴺ Φ(|r̄₁ - r̄ⱼ|)][∂w₂(x₁, x₁, t)/∂p̄₁]dxⱼ = (N/V)∫[∂Φ(|r̄₁ - r̄₂|)/∂r̄₁][∂w₂(x₁, x₂, t)/∂p̄₁]dx₂.

При проведении выкладок мы учли граничные условия w_N(x₁, ..., x_N, t)|_{p_α = ±∞} = 0, определение (13.1), а также то, что все частицы одинаковы и N - 1 ≈ N. Собирая в единую формулу найденные четыре выражения, получим первое уравнение цепочки уравнений ББГКИ.

Доказать, что в ходе эволюции замкнутой системы введённая Больцманом энтропия:

S₁ = -k∫w₁(r̄, p̄, t)ln w₁(r̄, p̄, t)dr̄dp̄

возрастает или остаётся неизменной.

Решение. Продифференцируем S₁ по времени:

dS₁/dt = -k∫(∂/∂t)(w₁ ln w₁)dx = -k∫[∂w₁/∂t ln w₁]dx - k∫w₁(1/w₁)(∂w₁/∂t)dx =     (13.20)
= -k(∂/∂t)∫w₁dx + k∫{w₁dx + k∫[(p̄/m)(∂w₁/∂r̄) - (∂U₀(r̄)/∂r̄)(∂w₁/∂p̄) - I]ln w₁dx.

Здесь I — интеграл столкновений (см. формулу (13.8)). При записи цепочки равенств мы учли, что три из четырёх слагаемых в dS₁/dt равны нулю.

Производная (∂/∂t)∫w₁dx = 0 в силу условия нормировки для функции w₁.

Чтобы убедиться в равенстве нулю интеграла ∫(p̄/m)(∂w₁/∂r̄)ln w₁dx проведём интегрирование по частям (по r̄), а затем воспользуемся граничными условиями w₁(r̄, p̄, t)|_{r_α = ±∞} = 0, где α = x, y, z. В результате получим:

k∫(p̄/m)(∂w₁/∂r̄)ln w₁dr̄dp̄ = k(p̄/m)w₁ ln w₁|_{r_α = -∞}^{r_α = ∞} dp̄ - k∫(p̄/m)w₁(∂ln w₁/∂r̄)dr̄dp̄ =
= -k∫(p̄/m)(∂w₁/∂r̄)dr̄dp̄ = -k∫(p̄/m)w₁|_{r_α = -∞}^{r_α = ∞} dp̄ = 0.

Интеграл ∫[∂U₀(r̄)/∂r̄](∂w₁/∂p̄)ln w₁dx также равен нулю. В этом можно убедиться, проведя интегрирование по частям (по p̄) и воспользовавшись граничными условиями w₁(r̄, p̄, t)|_{p_α = ±∞} = 0. Подставим в (13.20) интеграл столкновений в форме Н.Н. Боголюбова:

dS₁/dt = -(kN/mV)∫dp̄₂dp̄dr̄∫₀^∞ ρdρ∫₀^{2π} dφ|p̄₂ - p̄| ×
× [w₁(r̄, p̄₂', t)w₁(r̄, p̄', t) - w₁(r̄, p̄₂, t)w₁(r̄, p̄, t)] = J(t)     (13.21)

и проведём симметризацию подынтегрального выражения. Для этого представим J в виде суммы четырёх одинаковых слагаемых, каждое из которых равно J/4. Далее осуществим во втором и последующих заменами переменных: сначала p̄, p̄' на p̄₂, p̄₂', а затем p̄, p̄₂ на p̄', p̄₂'.

Учитывая равенства ρdρdφdp̄dp̄₂ = ρ'dρ'dφ'dp̄'dp̄₂' и являющиеся следствием уравнений движения, это позволяет записать (13.21) в виде:

dS₁/dt = (kN/4mV)∫dp̄₂dp̄dr̄∫₀^∞ ρdρ∫₀^{2π} dφ|p̄₂ - p̄| ×
× ln[w₁(r̄, p, t)w₁(r̄, p̄₂, t)/w₁(r̄, p̄', t)w₁(r̄, p̄₂', t)] × {w₁(r̄, p̄', t)w₁(r̄, p̄₂', t) - w₁(r̄, p̄, t)w₁(r̄, p̄₂, t)} ≥ 0.

Справедливость неравенства следует из того, что произведение логарифма на фигурную скобку в последнем подынтегральном выражении имеет структуру (B₁ - B₂)ln(B₁/B₂) и при любых положительных B₁ и B₂ положительно (равенство нулю возможно только при B₁ = B₂).

Пример 13.5

С помощью кинетического уравнения Больцмана с релаксационным членом (13.10) оценить величину коэффициента диффузии газа, концентрация которого изменяется вдоль оси 0x. Газ находится вблизи равновесного состояния, характеризующегося температурой T. Масса одной молекулы m, время релаксации τ, внешние поля отсутствуют, изменение концентрации считать стационарным.

Решение. В силу стационарности исследуемого процесса и из-за отсутствия внешних полей, первое и третье слагаемые в кинетическом уравнении Больцмана с релаксационным членом обращаются в ноль. Формула (13.10) после приведения оставшихся слагаемых к общему знаменателю примет вид:

w₁(r̄, p̄) = w₁^{(0)} - τ(p̄/m)∂w₁(r̄, p̄)/∂r̄,     (13.22)

где w₁^{(0)} — распределением Максвелла — Больцмана (см. (13.9)). u(x, t)n(x, t) = (N/V)∫(p_x/m)w₁(r̄, p̄)dp_xdp_ydp_z = (N/V)∫(p_x/m)w₁^{(0)}(r̄, p̄)dp_xdp_ydp_z -
- (τN/V)(∂/∂x)∫(p_x²/m²)w₁^{(0)}(x, p̄)dp_xdp_ydp_z = -(τkT/m)(∂n(x, t)/∂x).     (13.23)

При записи (13.23) учтено, что в равновесном газе поток частиц отсутствует и поэтому в соответствии с (13.19) (N/m)∫p_x²w₁^{(0)}(r̄, p̄)dp̄ = n(x)kT.

Первое и последнее выражения в цепочке равенств (13.23) совпадают соответственно с левой и правой частью уравнения (13.11), откуда коэффициент диффузии D = τkT/m.

Исходя из выражения (13.15) для потока тепла, возникающего при наличии градиента температуры, получить уравнение теплопроводности (13.16).

Решение. Выделим в среде небольшой параллелепипед, образованный плоскостями x = x₀, x = x₀ + Δx, y = y₀, y = y₀ + Δy, z = z₀, z = z₀ + Δz. Согласно (13.15) через поверхность x = x₀ за время dt в параллелепипед поступает количество теплоты dQ₁ = -ΔyΔzdtκ∂T/∂x|_{x=x₀}, а через поверхность x = x₀ + Δx уходит количество теплоты dQ₂ = -ΔyΔzdtκ∂T/∂x|_{x=x₀+Δx}. Аналогично записываются количества теплоты dQ₃, dQ₄, dQ₅ и dQ₆, поступающие или уходящие через другие четыре грани параллелепипеда.

Изменение температуры в малом объёме ΔxΔyΔz может происходить не только за счёт вышеупомянутых потоков теплоты, но и из-за наличия в нём тепловых источников q(x, y, z, t), генерирующих за время dt тепло q(x, y, z, t)ΔxΔyΔzdt. При этом уравнение теплового баланса принимает вид:

c_vρΔxΔyΔzdT = dQ₁ - dQ₂ + dQ₃ - dQ₄ + dQ₅ - dQ₆ + qΔxΔyΔzdt =
= κdt{Δy|Δz[(∂T/∂x)|_{x+Δx} - (∂T/∂x)|_x] + ΔxΔz[(∂T/∂y)|_{y+Δy} - (∂T/∂y)|_y] + ΔxΔy[(∂T/∂z)|_{z+Δz} - (∂T/∂z)|_z]} +
+ qΔxΔyΔzdt = κΔxΔyΔzdt(∂²T/∂x² + ∂²T/∂y² + ∂²T/∂z²) + qΔxΔyΔzdt,

откуда, после деления всех слагаемых на ΔxΔyΔzdt, получаем уравнение теплопроводности:

c_vρ(∂T/∂t) = κ(∂²T/∂x² + ∂²T/∂y² + ∂²T/∂z²) + q.

Его решение T(x, t), удовлетворяющее граничным условиям T(x = 0, t) = T₁(t), T(x = L, t) = T₂(t), линейно зависит от координаты x: T(x, t) = T₁(t) + x[T₂(t) - T₁(t)]/L. При этом поток тепла:

J_x = -κ∂T(x, t)/∂x = [T₂(t) - T₁(t)]/L     (13.24)

постоянен вдоль стержня, поэтому его можно выразить через dT₁,₂/dt:

J_x = -(c₁m₁/S)dT₁/dt = (c₂m₂/S)dT₂/dt.     (13.25)

Из (13.24), (13.25) легко получить уравнение для разности температур:

d[T₂(t) - T₁(t)]/dt = -(κS/L)(1/c₁m₁ + 1/c₂m₂)[T₂(t) - T₁(t)].

Его решение с учётом начального условия [T₂(t) - T₁(t)]|_{t=0} = (T₂₀ - T₁₀) имеет вид:

T₂(t) - T₁(t) = (T₂₀ - T₁₀)exp[-κSt(1/c₁m₁ + 1/c₂m₂)/L].
